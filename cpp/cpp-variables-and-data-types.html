Imagine you have a set of labeled containers in your kitchen pantry üì¶. You have one labeled "sugar," one labeled "flour," and another labeled "spices." You wouldn't put flour in the "sugar" container. In C++, variables are like these labeled containers, and data types are the labels. You must label a container with a specific type so the computer knows exactly what kind of information to store inside.

In C++, a variable is a named storage location in memory. Before you can use a variable, you must declare it by specifying its data type. This is a fundamental concept in C++ called static typing, where the type of data a variable holds is fixed. This helps the compiler manage memory efficiently and catch errors early.

C++ Data Types

  • int: For integers (whole numbers), like 10, -50, or 1000.
  • float: For single-precision floating-point numbers (decimals), like 3.14 or -0.5.
  • double: For double-precision floating-point numbers. They store more precise decimal values than float and are typically used for more accurate calculations.
  • char: For a single character, like 'A' or 'b'.
  • bool: For boolean values, which can only be true or false. These are used for logic and making decisions in your program.
  • std::string: A data type for storing sequences of characters (text), like "Hello, World!".
