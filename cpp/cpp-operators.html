ðŸ”¹ 3. Comparison Operators

Used to compare two values. The result is always true or false.

Operator Meaning Example Output
== Equal to 10 == 10 true
!= Not equal to 10 != 5 true
> Greater than 10 > 5 true
< Less than 5 < 10 true
>= Greater or equal 10 >= 10 true
<= Less or equal 5 <= 10 true

Real-life example:

  • Checking if your marks are equal to the passing score.
  • Checking if your age is greater than 18 to get a driver's license.

C++

#include 
int main() {
    int age = 20;
    std::cout << (age >= 18) << std::endl; // 1 (for true)
    std::cout << (age == 18) << std::endl; // 0 (for false)
    return 0;
}

ðŸ”¹ 4. Logical Operators

Used to combine true/false conditions.

Operator Meaning Example Output
&& Logical AND (both must be true) (5 > 2 && 10 > 5) true
|| Logical OR (at least one true) (5 > 10 || 10 > 5) true
! Logical NOT (reverses result) !(5 > 2) false

Real-life example:

  • A student passes if marks > 35 and attendance > 75.
  • You get a free delivery if order > $500 or you are a premium member.

C++

#include 
int main() {
    int marks = 80;
    int attendance = 90;
    std::cout << (marks > 35 && attendance > 75) << std::endl; // 1 (for true)
    return 0;
}

ðŸ”¹ 5. Bitwise Operators (Working with Bits)

Computers store numbers in bits (0s and 1s). Bitwise operators work on these bits.

Operator Meaning Example (a=5, b=3) Result
& AND a & b â†’ 101 & 011 1
| OR a | b â†’ 101 | 011 7
^ XOR a ^ b â†’ 101 ^ 011 6
~ NOT ~a â†’ ~101 -6
<< Left Shift a << 1 â†’ 1010 10
>> Right Shift a >> 1 â†’ 010 2

Real-life example:

  • Imagine two rows of light switches. & lights up only if the switches in both rows are ON.
  • << is like multiplying by 2, >> is like dividing by 2.

C++

#include 
int main() {
    int a = 5; // binary 101
    int b = 3; // binary 011
    std::cout << "a & b = " << (a & b) << std::endl; // 1
    std::cout << "a | b = " << (a | b) << std::endl; // 7
    std::cout << "a ^ b = " << (a ^ b) << std::endl; // 6
    return 0;
}

ðŸ”¹ 6. Pointer Operators

C++ has unique operators for working with memory addresses.

Operator Meaning Example
& Address-of &age gives the memory address of age
* Dereference *p gets the value at the address p points to

Real-life example:

Think of a book on a shelf. The & operator is like getting the shelf number and position of the book. The * operator is like going to that exact shelf and taking the book to read it.
