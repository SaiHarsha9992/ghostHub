When you send a text message to a friend üì±, you type your message and press "send," and your friend's phone displays it. In C++, the cout object works in a similar way: you put data into it, and it sends that data to be displayed on your computer's screen.

In C++, your first step is to make the computer display a message. While C used printf(), C++ introduced a more powerful and safer way to do this with the cout object. The cout object is part of the library, which handles input and output. Think of cout as an output stream, like a river carrying data to your screen. You use the << operator (called the stream insertion operator) to "push" data into this stream, and cout then displays it on the console.
