📂 Working with File System in Node.js – Read/Write/Streams

The File System (fs) module in Node.js is like the “USB drive of your code” — it lets your program read, write, and stream files directly on your computer. Think of it as the college lab assistant: always handling files, assignments, and saving your “all-nighter project” before the deadline 😅.

1. Importing the File System Module

The fs module comes built-in with Node.js (no npm install required 🎉). You can access it by requiring it like this:

const fs = require('fs');

That’s like saying, “Yo Node.js, bring me the file manager toolbox.”

2. Reading Files

Reading files in Node.js is like copying notes from your topper friend 📒. You have two main ways:

Synchronous Reading

const data = fs.readFileSync('example.txt', 'utf-8');
console.log(data);

This blocks everything until the file is read. Like waiting in the mess queue 🍛 — no other work until food arrives.

Asynchronous Reading

fs.readFile('example.txt', 'utf-8', (err, data) => {
  if (err) {
    console.log('Error reading file');
    return;
  }
  console.log(data);
});

Here, Node.js multitasks like your friend who eats biryani 🍗 while watching Netflix 📺.

3. Writing Files

Writing to a file is like submitting your assignment. You either overwrite or append content.

fs.writeFile('notes.txt', 'Hello, Node.js!', (err) => {
  if (err) throw err;
  console.log('File created successfully!');
});

Boom 💥 — new file created. Just don’t try this on your project files without backups (been there, cried later 🥲).

4. Working with Streams

Streams are like downloading a movie in parts instead of waiting for the whole 4GB file. Node.js processes data in chunks, making it memory-efficient.

const readStream = fs.createReadStream('bigfile.txt', 'utf-8');

readStream.on('data', (chunk) => {
  console.log('Received chunk:', chunk);
});

It’s like eating a family-sized pizza slice by slice 🍕 instead of trying to stuff the whole thing at once.

5. Real-World Analogy for Students

  • Read File: Copying your topper’s assignment 📑.
  • Write File: Submitting your own assignment.
  • Append File: Adding extra answers after the teacher announced bonus questions.
  • Streams: Sharing lecture notes on WhatsApp page by page instead of one huge PDF.

Always handle errors. Otherwise, your program crashes like your WiFi during online exams 😭.
