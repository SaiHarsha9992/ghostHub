In our daily life, we use symbols like +, -, *, / for calculations.

These are the operators you already know from school.

Operator Meaning Example Output
+ Addition 10 + 5 15
- Subtraction 10 - 5 5
* Multiplication 10 * 5 50
/ Division (decimal/int) 10 / 5 2
% Modulus (remainder) 10 % 3 1

üëâ Real-life example:

  • Adding prices of fruits in a bill.

üîπ 2. Assignment Operators

Assignment means putting a value into a variable.

Operator Example Same As
= x = 5 x = 5
+= x += 3 x = x + 3
-= x -= 2 x = x - 2
*= x *= 2 x = x * 2
/= x /= 2 x = x / 2

üëâ Real-life example:

  • Bank balance increases after deposit ‚Üí balance += amount.
  • Game score increases after hitting target ‚Üí score += 10.

score += 10;
System.out.println("Current Score: " + score); // 10

üîπ 3. Comparison (Relational) Operators

Used to compare two values. The result is always true or false.

Operator Meaning Example Output
== Equal to 10 == 10 true
!= Not equal to 10 != 5 true
> Greater than 10 > 5 true
< Less than 5 < 10 true
>= Greater or equal 10 >= 10 true
<= Less or equal 5 <= 10 true

üëâ Real-life example:

  • Checking if your marks are equal to pass marks.
  • Checking if your age is greater than 18.
int age = 20;
System.out.println(age >= 18); // true
System.out.println(age == 18); // false

üîπ 4. Logical Operators

Used to combine conditions.

Operator Meaning Example Output
&& AND (both true) (5 > 2 && 10 > 5) true
|| OR (one true) (5 > 10 || 10 > 2) true
! NOT (reverse) !(5 > 2) false

üëâ Real-life example:

  • A student passes if marks > 35 and attendance > 75.
  • You get free delivery if order > 500 or you are a premium member.
int marks = 80;
int attendance = 90;
System.out.println(marks > 35 && attendance > 75); // true

üîπ 5. Bitwise Operators (Working with bits)

Computers store numbers in bits (0s and 1s). Bitwise operators work on these bits.

Operator Meaning Example (a=5, b=3) Result
& AND 5 & 3 ‚Üí 101 & 011 1
| OR 5 | 3 ‚Üí 101 | 011 7
^ XOR 5 ^ 3 ‚Üí 101 ^ 011 6
~ NOT ~5 -6
<< Left Shift 5 << 1 10
>> Right Shift 5 >> 1 2

üëâ Real-life example:

  • & ‚Üí Two switches ON at the same time.
  • << ‚Üí Like multiplying by 2.
  • >> ‚Üí Like dividing by 2.
int a = 5; // binary 101
int b = 3; // binary 011
System.out.println(a & b); // 1
System.out.println(a | b); // 7

üîπ 6. Identity Operators (Not in Java)

In Python, is / is not check if two variables are the same object in memory.

üëâ In Java, we don‚Äôt have is. Instead:

  • Use == to compare primitive values (like int, char, boolean).
  • Use .equals() to compare objects (like Strings).
String x = new String("hello");
String y = new String("hello");

System.out.println(x == y); // false (different memory)
System.out.println(x.equals(y)); // true (same value)

üîπ 7. Membership Operators (Not in Java)

In Python, in / not in check if something exists in a sequence.

üëâ In Java, we do this using methods:

  • For strings ‚Üí contains()
  • For lists ‚Üí contains() from Collections
String word = "banana";
System.out.println(word.contains("na")); // true
System.out.println(!word.contains("z")); // true
