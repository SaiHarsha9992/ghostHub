üì¶ What is a Variable?

A variable is like a container (box) where you keep something for later use.

  • Example: You keep sugar in a jar and rice in a bag.
  • In Java, we keep data (like numbers or words) in a variable, but we must mention its type before storing it.

int age = 20;

Here:

  • name is a variable storing the word ‚Äúghost‚Äù.
  • age is a variable storing the number 20.

üî¢ Data Types in Java

Different types of data need different kinds of containers. Java gives us data types to store them.

Numbers (Integers & Floats)

  • Integers = whole numbers ‚Üí 10, -5, 100
  • Floating-point = decimal numbers ‚Üí 3.14, 0.99, -2.5
int x = 10; // integer
double pi = 3.14; // decimal number

Strings (Text)

Anything inside quotes " " is a string.

String name = "ghost";
String quote = "Learning Java is fun!";

Boolean (True/False)

Used for decisions.

boolean isStudent = true;
boolean isAdult = false;

(Optional) char (Single Character)

Stores a single letter, digit, or symbol.

char grade = 'A';
