Ternary Search

Part 1: Beginner's Guide to Ternary Search

What is Ternary Search?

Ternary search is a **search algorithm** that operates on **sorted arrays** or **unimodal functions**. Its core idea is to split the array into **three parts** and recursively reduce the search space.

Time Complexity: $O(\log_3 n) \approx O(\log n)$, which is comparable to binary search in practice.

Steps:

  1. Divide the array into 3 equal parts using two midpoints, mid1 and mid2.
  2. Compare the target with arr[mid1] and arr[mid2].
  3. Determine which segment contains the target and recurse into that segment.

Real-Life Example

Imagine finding a particular song in a sorted music playlist by duration. Instead of checking each song (linear search) or splitting in half (binary search), ternary search allows you to divide the playlist into three parts. This is efficient when playlists are huge, like 10,000+ songs.

Ternary Search in Programming Languages

C Example

#include <stdio.h>

int ternarySearch(int arr[], int l, int r, int key) {
    if (r >= l) {
        int mid1 = l + (r - l) / 3;
        int mid2 = r - (r - l) / 3;

        if (arr[mid1] == key) return mid1;
        if (arr[mid2] == key) return mid2;

        if (key < arr[mid1])
            return ternarySearch(arr, l, mid1 - 1, key);
        else if (key > arr[mid2])
            return ternarySearch(arr, mid2 + 1, r, key);
        else
            return ternarySearch(arr, mid1 + 1, mid2 - 1, key);
    }
    return -1;
}

int main() {
    int arr[] = {1, 3, 5, 7, 9, 11, 13};
    int n = sizeof(arr)/sizeof(arr[0]);
    int key = 7;
    int result = ternarySearch(arr, 0, n - 1, key);
    printf("Element found at index %d\n", result);
    return 0;
}

Python Example

def ternary_search(arr, l, r, key):
    if r >= l:
        mid1 = l + (r - l) // 3
        mid2 = r - (r - l) // 3

        if arr[mid1] == key:
            return mid1
        if arr[mid2] == key:
            return mid2

        if key < arr[mid1]:
            return ternary_search(arr, l, mid1 - 1, key)
        elif key > arr[mid2]:
            return ternary_search(arr, mid2 + 1, r, key)
        else:
            return ternary_search(arr, mid1 + 1, mid2 - 1, key)
    return -1

arr = [1, 3, 5, 7, 9, 11, 13]
key = 5
print("Element found at index", ternary_search(arr, 0, len(arr) - 1, key))

Java Example

class TernarySearch {
    static int ternarySearch(int l, int r, int key, int[] arr) {
        if (r >= l) {
            int mid1 = l + (r - l) / 3;
            int mid2 = r - (r - l) / 3;

            if (arr[mid1] == key) return mid1;
            if (arr[mid2] == key) return mid2;

            if (key < arr[mid1])
                return ternarySearch(l, mid1 - 1, key, arr);
            else if (key > arr[mid2])
                return ternarySearch(mid2 + 1, r, key, arr);
            else
                return ternarySearch(mid1 + 1, mid2 - 1, key, arr);
        }
        return -1;
    }

    public static void main(String[] args) {
        int[] arr = {1, 3, 5, 7, 9, 11, 13};
        int key = 9;
        int index = ternarySearch(0, arr.length - 1, key, arr);
        System.out.println("Element found at index " + index);
    }
}

Part 2: Key Points for Beginners

  • Works only on **sorted arrays**.
  • Divides the array into three parts, which can be faster than binary search in some cases.
  • Can also be used for **unimodal functions** (finding the maximum or minimum).
