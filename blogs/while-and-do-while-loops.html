While and Do While Loops

Introduction

Loops are fundamental to programming, allowing developers to execute a block of code multiple times based on certain conditions. Today, we’ll explore the while and do-while loops in four popular programming languages: C, C++, Java, and Python. We’ll break down their syntax and show you real-world examples to solidify your understanding.

1. The while Loop

The **while** loop runs as long as the given condition evaluates to true. If the condition is false initially, the loop will not run at all.

C & C++ Syntax:

while (condition) {
    // code to execute
}

Python Syntax:

while condition:
    # code to execute

Example: Counting from 1 to 5

C & C++ Example:

#include <stdio.h>
int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}

Java Example:

public class Main {
    public static void main(String[] args) {
        int i = 1;
        while (i <= 5) {
            System.out.println(i);
            i++;
        }
    }
}

Python Example:

i = 1
while i <= 5:
    print(i)
    i += 1

2. The do-while Loop

The **do-while** loop ensures that the block of code runs at least once, even if the condition is false initially. The condition is checked after the code execution.

C & C++ Syntax:

do {
    // code to execute
} while (condition);

Java Syntax:

do {
    // code to execute
} while (condition);

Python Syntax:

Note: Python does not have a direct do-while loop. However, we can simulate it using a while loop.

while True:
    # code to execute
    if not condition:
        break

Example: User Input Validation

C & C++ Example:

#include <stdio.h>
int main() {
    int age;
    do {
        printf("Enter your age: ");
        scanf("%d", &age);
    } while (age <= 0);
    printf("Your age is: %d\n", age);
    return 0;
}

Java Example:

import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int age;
        do {
            System.out.print("Enter your age: ");
            age = scanner.nextInt();
        } while (age <= 0);
        System.out.println("Your age is: " + age);
    }
}

Python Example:

age = 0
while age <= 0:
    age = int(input("Enter your age: "))
print(f"Your age is: {age}")

Real-World Use Cases

While Loop Example: ATM Simulation

C & C++ Example:

#include <stdio.h>
int main() {
    int pin = 1234, input;
    printf("Enter your PIN: ");
    while (scanf("%d", &input) && input != pin) {
        printf("Incorrect PIN. Try again: ");
    }
    printf("Access granted!\n");
    return 0;
}

Java Example:

import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        int pin = 1234, input;
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter your PIN: ");
        while ((input = scanner.nextInt()) != pin) {
            System.out.print("Incorrect PIN. Try again: ");
        }
        System.out.println("Access granted!");
    }
}

Python Example:

pin = 1234
input_pin = int(input("Enter your PIN: "))
while input_pin != pin:
    input_pin = int(input("Incorrect PIN. Try again: "))
print("Access granted!")

Do-While Loop Example: Retry Mechanism

You can use a do-while loop to prompt the user for input until they get it right (useful for retry mechanisms).

Key Differences Between while and do-while Loops

Feature while Loop do-while Loop
Condition Check Checked before the loop starts Checked after the loop starts
Execution May not execute if the condition is false initially Always executes at least once
Use Case Use when you’re unsure if the loop will run Use when the loop needs to run at least once

Conclusion

The while and do-while loops are fundamental to mastering control flow in programming. Understanding when and how to use each loop will help you write efficient, readable, and logical code. Whether validating input or running a process until a condition is met, these loops are your go-to tools.

By mastering these loops in C, C++, Java, and Python, you’re one step closer to becoming a proficient programmer.
