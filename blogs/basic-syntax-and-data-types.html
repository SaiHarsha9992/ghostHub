Basic Syntax and Data Types in C, C++, Python, and Java

Introduction

Understanding basic syntax and data types is the foundation of programming. Data types define the kind of data a variable can hold, such as numbers, characters, or boolean values. This guide will walk you through variables, constants, type conversion, and type casting with real-world examples in C, C++, Python, and Java.

1. Variables and Constants

What are Variables?

Variables are named storage locations in memory that hold data. Think of them as labeled jars in a kitchen pantry where you store different ingredients.

What are Constants?

Constants are fixed values that do not change during program execution. They are like the recipe instructions that remain the same every time you cook.

Real-Life Example:

Imagine a smart thermostat in your home. The current temperature is a **variable** because it changes throughout the day. The maximum allowed temperature (e.g., 24Â°C) is a **constant** because it remains fixed to ensure comfort.

Code Examples:

C: Variables and Constants

#include <stdio.h>
#define MAX_TEMP 24 // Constant

int main() {
    int currentTemp = 20; // Variable
    const float gravity = 9.8; // Constant
    printf("Current Temperature: %d\n", currentTemp);
    printf("Maximum Allowed Temperature: %d\n", MAX_TEMP);
    return 0;
}

C++: Variables and Constants

2. Data Types

Data types define what kind of values a variable can hold.

Common Data Types:

  • Integer: Whole numbers (e.g., 5, -10)
  • Float/Double: Decimal numbers (e.g., 3.14, -2.7)
  • Character: Single characters ('A', 'b')
  • String: Sequence of characters ("Hello")
  • Boolean: True/False values

Real-Life Example:

Imagine an online shopping website where:

  • The order ID is an **integer** (e.g., 12345).
  • The product price is a **float** (e.g., $19.99).
  • The customer's first initial is a **character** (e.g., 'J').
  • The product name is a **string** (e.g., "Wireless Mouse").
  • The payment status is a **boolean** (e.g., true for paid, false for pending).

Code Examples:

C: Data Types

#include <stdio.h>
int main() {
    int orderID = 12345;
    float price = 19.99;
    char initial = 'J';
    int paymentStatus = 1; // 1 for true (paid), 0 for false (pending)

    printf("Order ID: %d\n", orderID);
    printf("Price: %.2f\n", price);
    printf("Customer Initial: %c\n", initial);
    printf("Payment Status: %d\n", paymentStatus);

    return 0;
}

Python: Data Types

print(f"Customer Initial: {initial}")

3. Type Conversion and Type Casting

Type Conversion (Implicit and Explicit)

Implicit conversion: The compiler automatically converts smaller data types into larger ones.

Explicit conversion (Type Casting): The user manually converts one data type into another.

Real-Life Example:

Think about converting currency for international travel. If you're converting 100 Indian Rupees (INR) to U.S. Dollars (USD), the system automatically applies the exchange rate (implicit conversion). If you manually round off the converted amount to a whole number, that is explicit conversion.

Code Examples:

C: Type Conversion and Type Casting

#include <stdio.h>
int main() {
    int rupees = 100;
    float usd = rupees / 83.0; // Implicit conversion (int to float)
    int roundedUsd = (int) usd; // Explicit casting

    printf("Converted USD: %.2f\n", usd);
    printf("Rounded USD: %d\n", roundedUsd);

    return 0;
}

Python: Type Conversion and Type Casting

rupees = 100
usd = rupees / 83 # Implicit conversion
rounded_usd = int(usd) # Explicit casting

print(f"Converted USD: {usd:.2f}")

Java: Type Conversion and Type Casting

public class Main {
    public static void main(String[] args) {
        int rupees = 100;
        float usd = rupees / 83.0f; // Implicit conversion
        int roundedUsd = (int) usd; // Explicit casting

        System.out.println("Converted USD: " + usd);
        System.out.println("Rounded USD: " + roundedUsd);
    }
}
