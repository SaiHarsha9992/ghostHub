Part 1: Linear Search for Beginners

What is Linear Search?

Linear Search is the simplest searching technique. It goes through each element of a list one by one until it finds the target element or reaches the end of the list.

üëâ It does not require the list to be sorted.

  • Real-Life Example
  • Imagine you are searching for your friend in a crowded room. You go person by person, checking faces, until you find your friend. That‚Äôs linear search in action.

Algorithm (Step-by-Step)

  • Start from the first element.
  • Compare the target element with the current element.
  • If it matches ‚Üí return the position.
  • If not ‚Üí move to the next element.
  • If the end is reached without finding ‚Üí return "Not Found".

Pseudocode (Python)

LinearSearch(arr, target):
    for i from 0 to n-1:
        if arr[i] == target:
            return i
    return -1

Code Examples

‚úÖ C Code

#include 

int linearSearch(int arr[], int n, int target) {
    for(int i = 0; i < n; i++) {
        if(arr[i] == target)
            return i;
    }
    return -1;
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int n = sizeof(arr)/sizeof(arr[0]);
    int target = 30;

    int result = linearSearch(arr, n, target);
    if(result != -1)
        printf("Element found at index %d\n", result);
    else
        printf("Element not found\n");
    return 0;
}

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

arr = [5, 10, 15, 20, 25]
target = 20

result = linear_search(arr, target)

‚úÖ Java Code

public class LinearSearch {
    public static int linearSearch(int arr[], int target) {
        for(int i = 0; i < arr.length; i++) {
            if(arr[i] == target)
                return i;
        }
        return -1;
    }

    public static void main(String[] args) {
        int arr[] = {2, 4, 6, 8, 10};
        int target = 8;

        int result = linearSearch(arr, target);
        if(result != -1)
            System.out.println("Element found at index " + result);
        else
            System.out.println("Element not found");
    }
}

Complexity (Beginner View)

  • Best Case ‚Üí O(1) (target is at first position)
  • Worst Case ‚Üí O(n) (target not present or last element)
  • Space Complexity ‚Üí O(1)
