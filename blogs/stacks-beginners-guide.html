1. What is a Stack?

A Stack is a linear data structure that follows the principle: **LIFO** (Last In, First Out). The last element inserted is the first element to be removed.

üí° Real-Life Analogy:

  • **Plates in a cafeteria** üçΩÔ∏è: Place a plate on top (Push). Remove the top plate first (Pop).
  • **Undo/Redo in Word processors** ‚Ü©Ô∏è: The last action performed is the first one undone.

2. Why Learn Stacks?

Stacks are a fundamental concept and are used as a building block for many applications:

  • Expression evaluation (Infix to Postfix, etc.)
  • Balanced parentheses check (e.g., `({})`)
  • Browser history navigation (Back/Forward)
  • Recursion (the system's call stack)
  • Undo/Redo operations in text editors

3. Basic Stack Operations

Operation Description
Push(x) Adds an element `x` to the top of the stack.
Pop() Removes the top element from the stack.
Peek()/Top() Returns the top element without removing it.
isEmpty() Checks if the stack contains any elements.
isFull() (For fixed-size stacks) Checks if the stack is full.

4. Implementation of Stack

A. Using Arrays (Static Stack)

An array-based stack has a fixed size. We use a variable, `top`, to track the top of the stack.

C Example

#include 
#define MAX 5
int stack[MAX], top = -1;

void push(int x) {
    if (top == MAX-1) printf("Stack Overflow\n");
    else stack[++top] = x;
}

void pop() {
    if (top == -1) printf("Stack Underflow\n");
    else printf("Popped: %d\n", stack[top--]);
}

void peek() {
    if (top == -1) printf("Stack Empty\n");
    else printf("Top: %d\n", stack[top]);
}

int main() {
    push(10); push(20); push(30);
    peek(); 
    pop(); 
    peek();
    return 0;
}

Python Example

stack = []

stack.append(10) # Push
stack.append(20)
stack.append(30)

print("Top:", stack[-1]) # Peek
print("Popped:", stack.pop()) # Pop

Java Example

import java.util.Stack;

public class Main {
    public static void main(String[] args) {
        Stack stack = new Stack<>();
        stack.push(10);
        stack.push(20);
        stack.push(30);

        System.out.println("Top: " + stack.peek());
        System.out.println("Popped: " + stack.pop());
        System.out.println("Top after pop: " + stack.peek());
    }
}

B. Using Linked List (Dynamic Stack) ‚Äì Concept

Unlike an array, a linked list stack can grow dynamically. Each node in the list holds data and a pointer to the next node. The `push` operation involves inserting at the head, and `pop` involves removing from the head. (Code can be explored in an intermediate guide.)

5. Real-Time Examples

  • Undo/Redo in Text Editors ‚úçÔ∏è: Each action is pushed onto a stack. To undo, you simply pop the last action.
  • Browser Back/Forward üåê: The back button pops the current page from the history stack, while the forward button pushes it back onto a forward-history stack.
  • Recursive Function Calls üîÑ: The system manages a call stack to keep track of the sequence of function calls.
