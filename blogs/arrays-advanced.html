6. Find Duplicate Number (Floyd‚Äôs Tortoise & Hare)

üëâ Famous cycle detection trick.

Python: Find Duplicate Number

arr = [1,3,4,2,2]
slow = fast = arr[0]
while True:
    slow = arr[slow]
    fast = arr[arr[fast]]
    if slow == fast:
        break
slow = arr[0]
while slow != fast:
    slow = arr[slow]
    fast = arr[fast]
print("Duplicate Number:", slow)

Python: Median of Two Sorted Arrays

def findMedianSortedArrays(nums1, nums2):
    nums = sorted(nums1 + nums2)
    n = len(nums)
    if n % 2:
        return nums[n // 2]
    return (nums[n // 2 - 1] + nums[n // 2]) / 2

print(findMedianSortedArrays([1,3],[2]))

9. Subarray Sum Equals K

üëâ Uses prefix sum + hashmap.

Java: Subarray Sum Equals K

import java.util.*;
public class SubarraySum {
    public static void main(String[] args) {
        int[] arr = {1,2,3};
        int k = 3, sum = 0, count = 0;
        Map map = new HashMap<>();
        map.put(0,1);
        for (int num : arr) {
            sum += num;
            count += map.getOrDefault(sum - k, 0);
            map.put(sum, map.getOrDefault(sum, 0) + 1);
        }
        System.out.println("Count = " + count);
    }
}

10. Longest Consecutive Sequence

üëâ Uses HashSet for O(n).

C++: Longest Consecutive Sequence

#include 
#include 
#include 
using namespace std;
int main() {
    vector nums = {100,4,200,1,3,2};
    unordered_set s(nums.begin(), nums.end());
    int longest = 0;
    for (int num : s) {
        if (s.find(num - 1) == s.end()) {
            int curr = num, streak = 1;
            while (s.count(curr + 1)) {
                curr++;
                streak++;
            }
            longest = max(longest, streak);
        }
    }
    cout << "Longest Consecutive Sequence = " << longest;
    return 0;
}
