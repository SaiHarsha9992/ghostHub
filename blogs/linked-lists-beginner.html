Linked Lists

Part 1: Linked Lists for Beginners

1. Introduction: What is a Linked List?

A Linked List is a fundamental data structure where each element (called a **node**) holds two things: the data and a reference (or pointer) to the next node.

Unlike arrays, which are continuous blocks of memory, linked lists are dynamic. This means you can easily add or remove elements without worrying about shifting memory, making them very flexible in size.

2. Real-Life Analogy

Imagine a train üöÜ:

  • Each coach is a **node**.
  • A coach carries passengers (the **data**).
  • It has a connector (the **pointer**) that links to the next coach.

If you want to add a coach, you just connect it to the chain. To remove one, you simply disconnect it. That‚Äôs a Linked List in real life.

3. Why Use Linked Lists?

  • **Dynamic Size:** They can grow or shrink at runtime without needing a fixed size.
  • **Efficient Insertions/Deletions:** Adding or removing elements is fast, as it only requires changing a few pointers and no shifting of elements like in an array.
  • **Flexibility:** Nodes can be stored anywhere in memory, as long as the links are correctly maintained.

This example demonstrates how to create a basic singly linked list with three nodes in C.

struct Node {
    int data;
    struct Node* next;
};

int main() {
    struct Node* head = NULL;
    struct Node* second = NULL;
    struct Node* third = NULL;

    head = (struct Node*)malloc(sizeof(struct Node));
    second = (struct Node*)malloc(sizeof(struct Node));
    third = (struct Node*)malloc(sizeof(struct Node));

    head->data = 10;
    head->next = second;

    second->data = 20;
    second->next = third;

    third->data = 30;
    third->next = NULL;

    struct Node* temp = head;
    while(temp != NULL) {
        printf("%d -> ", temp->data);
        temp = temp->next;
    }
    return 0;
}

6. Types of Linked Lists (Basic)

  • **Singly Linked List:** Nodes are connected in a single direction.
  • **Doubly Linked List:** Nodes have pointers to both the next and previous nodes.
  • **Circular Linked List:** The last node's pointer connects back to the first node, forming a loop.

7. Real-Time Applications

  • **Music Playlist:** You can easily move to the next or previous song.
  • **Undo/Redo:** Each action can be a node in a list.
  • **Browser History:** Navigating back and forth between pages.
  • **Dynamic Memory Allocation:** Used in operating systems to manage memory.
