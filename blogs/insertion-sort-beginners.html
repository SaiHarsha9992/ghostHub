Insertion Sort

Part 1 – Insertion Sort for Beginners

What is Insertion Sort?

Insertion Sort works like the way you arrange playing cards in your hand.

  • You pick one card at a time.
  • You place it in the correct position among the already sorted cards.
  • Repeat until all cards are sorted.

This makes Insertion Sort intuitive and simple.

Real-Life Example

Imagine you’re arranging books on a shelf by height:

  • Start with the first book (it's already "sorted").
  • Pick the next book and place it in the correct position among the previous ones.
  • Keep repeating until the entire shelf is sorted.

This is exactly how insertion sort works!

Algorithm (Step-by-Step)

  1. Start with the second element (index 1).
  2. Compare it with the elements before it.
  3. Insert it into its correct position by shifting other elements.
  4. Repeat this process for each element until the entire array is sorted.

Time Complexity

  • Best Case (Already Sorted): $O(n)$
  • Worst Case (Reverse Order): $O(n^2)$
  • Average Case: $O(n^2)$
  • Space Complexity: $O(1)$ (In-place sorting)

Code Examples

C Program

#include 

void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

int main() {
    int arr[] = {12, 11, 13, 5, 6};
    int n = sizeof(arr)/sizeof(arr[0]);
    insertionSort(arr, n);
    for (int i = 0; i < n; i++)
        printf("%d ", arr[i]);
    return 0;
}

Java Program

public class InsertionSort {
    public static void insertionSort(int arr[]) {
        for (int i = 1; i < arr.length; i++) {
            int key = arr[i];
            int j = i - 1;
            while (j >= 0 && arr[j] > key) {
                arr[j + 1] = arr[j];
                j--;
            }
            arr[j + 1] = key;
        }
    }

    public static void main(String[] args) {
        int arr[] = {12, 11, 13, 5, 6};
        insertionSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}

Key Takeaways (Beginners)

  • Simple and **intuitive** sorting method.
  • Good for **small datasets** or **partially sorted** arrays.
  • Works **in-place**, making it memory efficient.
