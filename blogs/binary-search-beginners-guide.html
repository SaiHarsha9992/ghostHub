1. What is Binary Search?

Binary Search is an efficient searching algorithm used to find an element in a **sorted** list/array. Instead of checking each element one by one (like Linear Search), it divides the array into halves, eliminating half of the search space at each step.

Time Complexity: O(log n) vs O(n) for linear search.

Imagine you‚Äôre looking for a word in a dictionary:

  • You don‚Äôt start from the first page.
  • Instead, you open the middle, check if the word is before or after that page, and repeat until you find it. That‚Äôs binary search in action!
  • Another example:
  • üî¢ Guessing a number between 1‚Äì100 with "higher/lower" hints ‚Üí you halve the range each time.

3. How Binary Search Works (Step-by-Step)

Given a sorted array:

[10, 20, 30, 40, 50, 60, 70]

Target = 50

  • Start with low = 0, high = n-1.
  • Find mid = (low + high) / 2.
  • If arr[mid] == target, found! üéâ
  • If arr[mid] < target, search right half.
  • If arr[mid] > target, search left half.
  • Repeat until found or low > high (not present).

4. Code Examples

C

#include 
int binarySearch(int arr[], int n, int key) {
    int low = 0, high = n - 1;
    while (low <= high) {
        int mid = (low + high) / 2;
        if (arr[mid] == key) return mid;
        else if (arr[mid] < key) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}
int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int n = sizeof(arr)/sizeof(arr[0]);
    int key = 40;
    int result = binarySearch(arr, n, key);
    if (result != -1) printf("Found at index %d", result);
    else printf("Not found");
}

def binary_search(arr, key):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == key:
            return mid
        elif arr[mid] < key:
            low = mid + 1
        else:
            high = mid - 1
    return -1

public class BinarySearch {
    public static int binarySearch(int[] arr, int key) {
        int low = 0, high = arr.length - 1;
        while (low <= high) {
            int mid = (low + high) / 2;
            if (arr[mid] == key) return mid;
            else if (arr[mid] < key) low = mid + 1;
            else high = mid - 1;
        }
        return -1;
    }
    public static void main(String[] args) {
        int[] arr = {10,20,30,40,50};
        int key = 20;
        int result = binarySearch(arr, key);
        if (result != -1) System.out.println("Found at index " + result);
        else System.out.println("Not found");
    }
}

5. Common Beginner Mistakes üö©

  • ‚ùå Forgetting that the array must be sorted.
  • ‚ùå Using (low + high)/2 which can overflow in big arrays (fix: low + (high-low)/2).
  • ‚ùå Not updating low or high correctly ‚Üí leads to infinite loop.
