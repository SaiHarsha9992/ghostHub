2. Advanced Types of Linked Lists

Beyond the simple singly linked list, there are two common variations:

Doubly Linked List (DLL)

In a doubly linked list, each node has a reference to both the **next** and the **previous** node. This makes it useful for bidirectional traversal, like a back and forward button in a browser.

C++ Implementation

class Node {
public:
    int data;
    Node* next;
    Node* prev;
    Node(int val) {
        data = val;
        next = prev = NULL;
    }
};

Circular Linked List (CLL)

In a circular linked list, the last node's `next` pointer connects back to the first node, forming a loop. This is useful in scenarios like round-robin scheduling for CPU tasks.

3. Key Advanced Operations

Mastering these operations is crucial for using linked lists effectively:

  • Insertion at the beginning, end, or middle of the list.
  • Deletion by a given value or position.
  • Reversing a Linked List.
  • Detecting a cycle (e.g., using **Floyd‚Äôs Cycle Detection Algorithm** with fast and slow pointers).
  • Merging two sorted linked lists.
  • Finding the middle element of a list in $O(n)$ time using two pointers (one slow, one fast).

Example: Reverse a Linked List (C++)

Node* reverse(Node* head) {
    Node* prev = NULL;
    Node* curr = head;
    Node* next = NULL;

    while(curr != NULL) {
        next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}

4. Real-Life Advanced Examples

  • **Undo/Redo:** The feature in applications like MS Word is often implemented with a Doubly Linked List to quickly traverse back and forth.
  • **Round-robin scheduling:** Used in operating systems, this is a classic application for a Circular Linked List.
  • **LRU Cache:** A Least Recently Used (LRU) Cache is often implemented using a combination of a Doubly Linked List and a HashMap for efficient access and updates.
