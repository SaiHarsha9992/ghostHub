Queues – Beginner Level

1. What is a Queue?

A Queue is a linear data structure that follows the principle of **FIFO** (First In, First Out). The first element added is the first one to be removed.

💡 Analogy: Think of a queue at a ticket counter:

  • The first person in line is served first.
  • New people join at the end of the line.
  • You can’t skip the line!

Properties of a Queue:

  • **Enqueue:** Inserts an element at the rear.
  • **Dequeue:** Removes an element from the front.
  • **Peek/Front:** Views the first element without removing it.
  • **IsEmpty/IsFull:** Checks the current state of the queue.

2. Why Learn Queues?

Queues are a fundamental concept with a wide range of applications:

  • **CPU scheduling** in operating systems.
  • Managing **printer queues**.
  • **Call center** systems.
  • **Breadth-First Search (BFS)** in graphs.
  • **Message passing** in distributed systems.

3. Queue Operations (Beginner Level)

  1. **Enqueue (Insert):** Adds an element to the back of the queue.
  2. **Dequeue (Remove):** Removes and returns the element from the front of the queue.
  3. **Peek (Front element):** Returns the element at the front of the queue without removing it.
  4. **Display (Traverse):** Prints all elements in the queue.

4. Code Examples

Here are implementations of a simple queue using arrays and lists.

C – Simple Queue Implementation

#include 
#define SIZE 5

int queue[SIZE], front = -1, rear = -1;

void enqueue(int val) {
    if (rear == SIZE - 1) {
        printf("Queue Overflow\n");
    } else {
        if (front == -1) front = 0;
        queue[++rear] = val;
    }
}

void dequeue() {
    if (front == -1 || front > rear) {
        printf("Queue Underflow\n");
    } else {
        printf("Dequeued: %d\n", queue[front++]);
    }
}

void display() {
    if (front == -1 || front > rear) {
        printf("Queue is empty\n");
    } else {
        for (int i = front; i <= rear; i++)
            printf("%d ", queue[i]);
        printf("\n");
    }
}

int main() {
    enqueue(10);
    enqueue(20);
    enqueue(30);
    display();
    dequeue();
    display();
    return 0;
}

C++ – Using Arrays

#include 
using namespace std;
#define SIZE 5

class Queue {
    int arr[SIZE], front, rear;
public:
    Queue() { front = -1; rear = -1; }

    void enqueue(int val) {
        if (rear == SIZE - 1) cout << "Queue Overflow\n";
        else {
            if (front == -1) front = 0;
            arr[++rear] = val;
        }
    }

    void dequeue() {
        if (front == -1 || front > rear) cout << "Queue Underflow\n";
        else cout << "Dequeued: " << arr[front++] << endl;
    }

    void display() {
        if (front == -1 || front > rear) cout << "Queue Empty\n";
        else {
            for (int i = front; i <= rear; i++) cout << arr[i] << " ";
        }
        cout << endl;
    }
};

int main() {
    Queue q;
    q.enqueue(10);
    q.enqueue(20);
    q.enqueue(30);
    q.display();
    q.dequeue();
    q.display();
    return 0;
}

Java – Queue with Array

class Queue {
    int front, rear, size;
    int arr[];

    Queue(int capacity) {
        front = rear = -1;
        size = capacity;
        arr = new int[size];
    }

    void enqueue(int val) {
        if (rear == size - 1) {
            System.out.println("Queue Overflow");
        } else {
            if (front == -1) front = 0;
            arr[++rear] = val;
        }
    }

    void dequeue() {
        if (front == -1 || front > rear) {
            System.out.println("Queue Underflow");
        } else {
            System.out.println("Dequeued: " + arr[front++]);
        }
    }

    void display() {
        if (front == -1 || front > rear) {
            System.out.println("Queue Empty");
        } else {
            for (int i = front; i <= rear; i++) {
                System.out.print(arr[i] + " ");
            }
            System.out.println();
        }
    }

    public static void main(String[] args) {
        Queue q = new Queue(5);
        q.enqueue(10);
        q.enqueue(20);
        q.enqueue(30);
        q.display();
        q.dequeue();
        q.display();
    }
}

5. Real-Life Example of Queues

Queues are everywhere in our daily lives:

  • **Bank Queue:** Customers are served in the order they arrive.
  • **Printer Queue:** The first document sent to the printer is printed first.
  • **Call Center:** The first caller in line gets the next available agent.
  • **Bus Queue:** The first person in the queue gets on the bus first.

6. Problem-Solving Strategies (Beginner)

  • Always check for **overflow** and **underflow** conditions before performing `enqueue` or `dequeue`.
  • Remember the **FIFO** principle. A queue is like a line at a shop.
  • Use `front` and `rear` pointers to keep track of the start and end of the queue.
