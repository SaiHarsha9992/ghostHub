üîπ Advanced String Handling in C

In C, strings are character arrays, so advanced operations require manual handling.

1. Tokenization (strtok)

Used to split strings into tokens (words).

#include 
#include 

int main() {
    char str[] = "Hello,World,Strings,In,C";
    char *token = strtok(str, ",");

    while (token != NULL) {
        printf("Token: %s\n", token);
        token = strtok(NULL, ",");
    }
    return 0;
}

2. String Formatting (sprintf)

#include 

int main() {
    char buffer[100];
    int age = 20;
    sprintf(buffer, "I am %d years old", age);
    printf("%s\n", buffer);
    return 0;
}

3. Dynamic Strings

Since C has no dynamic strings, we use malloc() and realloc().

#include 
#include 
#include 

int main() {
    char *str = malloc(20);
    strcpy(str, "Hello");
    str = realloc(str, 40);
    strcat(str, " World");
    printf("%s\n", str);
    free(str);
    return 0;
}

üîπ Advanced String Handling in C++

C++ offers rich advanced features via std::string and stringstream.

1. String Streams (stringstream)

Great for parsing and formatting.

#include 
#include

üîπ Advanced String Handling in Python

Python is extremely powerful with advanced string manipulation.

1. Regular Expressions (re module)

import re

text = "Contact: user@example.com, phone: +91-9876543210"

# Find email
email = re.findall(r"\w+@\w+\.\w+", text)
print("Emails:", email)

# Find phone
phone = re.findall(r"\+\d{2}-\d{10}", text)
print("Phones:", phone)

2. String Formatting (f-strings, .format())

name = "Harsha"
age = 20

print(f"My name is {name}, I am {age} years old")
print("My name is {}, I am {} years old".format(name, age))

3. Raw Strings

Useful for regex paths (r"...").

path = r"C:\Users\Harsha\Desktop"
print(path)  # C:\Users\Harsha\Desktop

4. Mutable Strings (list conversion / io.StringIO)

from io import StringIO

s = StringIO("Hello")
s.write(" World")
print(s.getvalue())

üîπ Advanced String Handling in Java

Java provides powerful libraries for advanced string operations.

1. Regular Expressions (Pattern, Matcher)

import java.util.regex.*;

public class RegexExample {
    public static void main(String[] args) {
        String text = "Email: test@example.com";
        Pattern p = Pattern.compile("\\w+@\\w+\\.\\w+");
        Matcher m = p.matcher(text);

        if (m.find()) {
            System.out.println("Found: " + m.group());
        }
    }
}

2. StringBuilder (Mutable Strings)

String in Java is immutable. For efficiency, use StringBuilder.

public class BuilderExample {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder("Hello");
        sb.append(" World");
        sb.insert(5, " Beautiful");
        sb.delete(5, 15);
        System.out.println(sb);
    }
}

3. String Formatting

public class FormatExample {
    public static void main(String[] args) {
        String name = "Harsha";
        int age = 20;

        String result = String.format("My name is %s and I am %d years old", name, age);
        System.out.println(result);
    }
}

4. Tokenization (String.split(), StringTokenizer)

import java.util.StringTokenizer;

public class TokenExample {
    public static void main(String[] args) {
        String text = "Hello,World,Java";
        StringTokenizer st = new StringTokenizer(text, ",");

        while (st.hasMoreTokens()) {
            System.out.println("Token: " + st.nextToken());
        }
    }
}
