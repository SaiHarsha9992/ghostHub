The for Loop: Repeating with Control

General Syntax:

Example 1: Printing Numbers from 1 to 5

Python Example:

for i in range(1, 6):
    print(i)

C Example:

#include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

C++ Example:

#include <iostream>
using namespace std;
int main() {
    for (int i = 1; i <= 5; i++) {
        cout << i << endl;
    }
    return 0;
}

Java Example:

public class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }
    }
}

Example 2: Summing Numbers (1 to 10)

Python Example:

total = 0
for i in range(1, 11):
    total += i
print("Sum =", total)

C Example:

#include <stdio.h>
int main() {
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        sum += i;
    }
    printf("Sum = %d\n", sum);
    return 0;
}

C++ Example:

#include <iostream>
using namespace std;
int main() {
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        sum += i;
    }
    cout << "Sum = " << sum << endl;
    return 0;
}

Java Example:

public class Main {
    public static void main(String[] args) {
        int sum = 0;
        for (int i = 1; i <= 10; i++) {
            sum += i;
        }
        System.out.println("Sum = " + sum);
    }
}

Example 3: Iterating Over a Collection

Python Example:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

C Example (using arrays):

#include <stdio.h>
int main() {
    char *fruits[] = {"apple", "banana", "cherry"};
    for (int i = 0; i < 3; i++) {
        printf("%s\n", fruits[i]);
    }
    return 0;
}

C++ Example:

#include <iostream>
using namespace std;
int main() {
    string fruits[] = {"apple", "banana", "cherry"};
    for (int i = 0; i < 3; i++) {
        cout << fruits[i] << endl;
    }
    return 0;
}

Java Example:

public class Main {
    public static void main(String[] args) {
        String[] fruits = {"apple", "banana", "cherry"};
        for (int i = 0; i < fruits.length; i++) {
            System.out.println(fruits[i]);
        }
    }
}

Bonus: Enhanced For Loops

Some languages have a simpler syntax for looping directly through collections.

Python Example:

for fruit in fruits:
    print(fruit)

C++ (range-based loop):

for (string fruit : fruits) {
    cout << fruit << endl;
}

Java (enhanced for loop):

for (String fruit : fruits) {
    System.out.println(fruit);
}
