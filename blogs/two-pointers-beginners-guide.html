2. What is the Two Pointers Technique?

The Two Pointers method uses two variables (pointers) that move across an array or string in a coordinated way to solve a problem.

  • One pointer starts from the left.
  • The other pointer starts from the right.
  • Depending on the logic, they either move towards each other or move in the same direction.

3. Syntax (General Pseudocode)

left = 0
right = n-1

while left < right:
    if condition satisfied:
        do something
    else:
        move left/right pointer

5. Beginner Example: Check if a String is a Palindrome

Problem: A string is a palindrome if it reads the same forward and backward.

bool isPalindrome(char s[]) {
    int left = 0, right = strlen(s) - 1;
    while (left < right) {
        if (s[left] != s[right]) return false;
        left++;
        right--;
    }
    return true;
}

int main() {
    char str[] = "madam";

    return 0;
}

Python Program

def is_palindrome(s):
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True

print("Palindrome" if is_palindrome("racecar") else "Not Palindrome")

Instead of a brute-force approach that might check each possible substring ($O(n^2)$), this solution is solved in linear time, $O(n)$, using two pointers.
