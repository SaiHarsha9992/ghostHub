Selection Sort

Part 1: Selection Sort for Beginners

What is Selection Sort?

Selection Sort is a comparison-based sorting algorithm. It repeatedly selects the smallest (or largest) element from the unsorted portion of the array and places it in its correct position.

Time complexity:

  • Best case: $O(n^2)$
  • Worst case: $O(n^2)$
  • Space complexity: $O(1)$ (in-place sorting)

Real-Life Example of Selection Sort

Imagine you are arranging books on a shelf by height:

  1. You scan the entire shelf and find the smallest book.
  2. You place it in the first position.
  3. Then, from the remaining books, you find the next smallest and place it second.
  4. You repeat this process until all books are sorted.

This manual "picking the smallest and placing it" is exactly what Selection Sort does.

Step-by-Step Working of Selection Sort

Let’s say we have an array: [29, 10, 14, 37, 13]

  • Find the smallest element in the entire array → 10. Swap it with the first element (29) → [10, 29, 14, 37, 13].
  • From the remaining unsorted portion ([29, 14, 37, 13]), the smallest is 13. Swap it with the first element of this portion (29) → [10, 13, 14, 37, 29].
  • The next smallest is 14, which is already in its correct position.
  • The next smallest from the remaining part ([37, 29]) is 29. Swap it with 37 → [10, 13, 14, 29, 37].

The array is now sorted! ✅

Code Examples

C Implementation

#include 
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIndex = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex])
                minIndex = j;
        }
        int temp = arr[minIndex];
        arr[minIndex] = arr[i];
        arr[i] = temp;
    }
}
int main() {
    int arr[] = {29, 10, 14, 37, 13};
    int n = sizeof(arr)/sizeof(arr[0]);
    selectionSort(arr, n);
    for (int i = 0; i < n; i++) printf("%d ", arr[i]);
    return 0;
}

void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIndex = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) minIndex = j;

    }
}

Python Implementation

def selection_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        min_index = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

arr = [29, 10, 14, 37, 13]
print(selection_sort(arr))

Java Implementation

public class SelectionSort {
    static void selectionSort(int arr[]) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }
    public static void main(String[] args) {
        int arr[] = {29, 10, 14, 37, 13};
        selectionSort(arr);
        for (int num : arr) System.out.print(num + " ");
    }
}

Advantages & Disadvantages

  • ✅ **Simple** and easy to implement.
  • ✅ Works well on **small datasets**.
  • ❌ Inefficient for large datasets ($O(n^2)$).
  • ❌ **Not adaptive** (does not get faster if partially sorted).
