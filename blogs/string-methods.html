üîπ String Methods in C

In C, strings are character arrays (char[]) terminated with \0. Since C doesn‚Äôt have a built-in string class, most operations are provided by library.

Common Methods in C:

  • strlen() ‚Üí Get length of a string
  • strcpy() ‚Üí Copy one string into another
  • strcat() ‚Üí Concatenate (append) strings
  • strcmp() ‚Üí Compare two strings
  • strchr() ‚Üí Find first occurrence of a character
  • strstr() ‚Üí Find a substring

Example:

#include 
#include 
#include 

int main() {
    char str1[50] = "Hello";
    char str2[] = " World";
    char str3[50];

    // Length
    printf("Length of str1: %lu\n", strlen(str1));

    // Copy
    strcpy(str3, str1);
    printf("Copy str1 into str3: %s\n", str3);

    // Concatenate
    strcat(str1, str2);
    printf("Concatenation: %s\n", str1);

    // Compare
    if (strcmp(str1, str3) == 0)
        printf("str1 and str3 are equal\n");
    else
        printf("str1 and str3 are different\n");

    // Search character
    char *ptr = strchr(str1, 'W');
    if (ptr)
        printf("Found 'W' at position: %ld\n", ptr - str1);

    // Search substring
    char *sub = strstr(str1, "World");
    if (sub)
        printf("Substring 'World' found!\n");

    return 0;
}

üîπ String Methods in C++

In C++, we have the std::string class from . It provides member functions that make string manipulation easier.

Common Methods in C++:

  • .length() or .size() ‚Üí Length of string
  • .append() or + ‚Üí Concatenate
  • .substr(start, len) ‚Üí Extract substring
  • .find("sub") ‚Üí Search substring
  • .compare() ‚Üí Compare strings
  • .c_str() ‚Üí Convert string to char*
  • .erase() ‚Üí Remove characters
  • .insert(pos, str) ‚Üí Insert substring

Example:

#include 
#include 
#include 
using namespace std;

int main() {
    string str1 = "Hello";
    string str2 = " World";

    // Length
    cout << "Length of str1: " << str1.length() << endl;

    // Concatenation
    string result = str1 + str2;
    cout << "Concatenation: " << result << endl;

    // Substring
    cout << "Substring (0-5): " << result.substr(0, 5) << endl;

    // Search
    size_t pos = result.find("World");
    if (pos != string::npos)
        cout << "'World' found at position: " << pos << endl;

    // Compare
    if (str1.compare("Hello") == 0)
        cout << "str1 equals 'Hello'" << endl;

    // Insert
    result.insert(5, " Beautiful");
    cout << "After Insert: " << result << endl;

    // Erase
    result.erase(5, 10);
    cout << "After Erase: " << result << endl;

    return 0;
}

üîπ String Methods in Python

Python makes string manipulation super easy. Strings are immutable, but Python provides a huge set of methods.

Common Methods in Python:

  • len() ‚Üí Length
  • .upper() / .lower() / .title() ‚Üí Case conversion
  • .strip() ‚Üí Remove spaces
  • .replace(old, new) ‚Üí Replace substring
  • .find("sub") / .index("sub") ‚Üí Search substring
  • .split(delimiter) ‚Üí Split string into list
  • .join(list) ‚Üí Join list into string
  • .startswith() / .endswith() ‚Üí Check prefixes/suffixes

Example:

str1 = "  Hello World  "

# Length
print("Length:", len(str1))

# Strip whitespace
print("Strip:", str1.strip())

# Upper & Lower
print("Uppercase:", str1.upper())
print("Lowercase:", str1.lower())

# Replace
print("Replace:", str1.replace("World", "Python"))

# Find
print("Find 'World':", str1.find("World"))

# Split & Join
words = str1.split()
print("Split:", words)
print("Join:", "-".join(words))

# Prefix/Suffix check
print("Starts with 'He':", str1.strip().startswith("He"))
print("Ends with 'ld':", str1.strip().endswith("ld"))

üîπ String Methods in Java

In Java, the String class provides powerful methods for manipulation. Strings are immutable.

Common Methods in Java:

  • .length() ‚Üí Length of string
  • .toUpperCase() / .toLowerCase() ‚Üí Case conversion
  • .trim() ‚Üí Remove leading/trailing spaces
  • .substring(start, end) ‚Üí Extract substring
  • .charAt(index) ‚Üí Get character
  • .indexOf("sub") / .lastIndexOf() ‚Üí Find substring
  • .replace("old", "new") ‚Üí Replace substring
  • .split(" ") ‚Üí Split string into array
  • .equals() / .equalsIgnoreCase() ‚Üí Compare

Example:

public class StringMethods {
    public static void main(String[] args) {
        String str1 = "  Hello World  ";

        // Length
        System.out.println("Length: " + str1.length());

        // Trim
        System.out.println("Trim: '" + str1.trim() + "'");

        // Case conversion
        System.out.println("Uppercase: " + str1.toUpperCase());
        System.out.println("Lowercase: " + str1.toLowerCase());

        // Substring
        System.out.println("Substring (2-7): " + str1.substring(2, 7));

        // Character at
        System.out.println("Char at 2: " + str1.charAt(2));

        // Search
        System.out.println("Index of 'World': " + str1.indexOf("World"));

        // Replace
        System.out.println("Replace: " + str1.replace("World", "Java"));

        // Split
        String[] words = str1.trim().split(" ");
        for (String w : words) {
            System.out.println("Word: " + w);
        }

        // Comparison
        System.out.println("Equals 'Hello World'? " + str1.trim().equals("Hello World"));
    }
}
