Optimized Bubble Sort

We can optimize Bubble Sort by checking if any swaps occurred during a pass. If no elements were swapped, it means the array is already sorted, and we can stop the algorithm early. This significantly improves the best-case time complexity.

Code Implementations

Here are implementations of the optimized Bubble Sort algorithm in different programming languages:

Python Code

def bubble_sort_optimized(arr):
    n = len(arr)
    for i in range(n - 1):
        swapped = False
        for j in range(n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break

# Example usage
arr = [5, 3, 8, 4, 2]
bubble_sort_optimized(arr)
print(arr)

C++ Code

using namespace std;

void bubbleSortOptimized(int arr[], int n) {
    bool swapped;
    for (int i = 0; i < n - 1; i++) {
        swapped = false;
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        if (swapped == false) {
            break;
        }
    }
}

int main() {
    int arr[] = {5, 3, 8, 4, 2};
    int n = sizeof(arr) / sizeof(arr[0]);
    bubbleSortOptimized(arr, n);
    for (int i = 0; i < n; i++)
        cout << arr[i] << " ";
    return 0;
}

Java Code

public class OptimizedBubbleSort {
    public static void bubbleSortOptimized(int arr[]) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            boolean swapped = false;
            for (int j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                    swapped = true;
                }
            }
            if (!swapped) {
                break;
            }
        }
    }

    public static void main(String args[]) {
        int arr[] = {5, 3, 8, 4, 2};
        bubbleSortOptimized(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}

Complexity Analysis

  • Best Case (Already Sorted): $O(n)$ with the optimization
  • Worst Case (Reverse Sorted): $O(n^2)$
  • Average Case: $O(n^2)$
  • Space Complexity: $O(1)$ (in-place sorting)
  • Stability: Stable (does not change the relative order of equal elements)

Variations

  • Cocktail Shaker Sort (Bidirectional Bubble Sort): Swaps in both directions, from start to end and then from end to start.
  • Odd-Even Sort (Parallel Bubble Sort): Designed to work on parallel computing systems.

When to Use Bubble Sort?

  • For educational purposes to learn fundamental sorting concepts.
  • When the input size is very small (e.g., $n < 20$).
  • When the array is almost sorted, as the optimized version will be very fast.

For large datasets, it is much more efficient to use advanced algorithms like Merge Sort, Quick Sort, or Heap Sort.
