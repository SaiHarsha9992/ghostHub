Mathematics in Programming

Programming and mathematics go hand-in-hand. For example, the Fuel Efficiency Calculator combines arithmetic and variables.

Code Examples:

C: Fuel Efficiency

#include 
int main() {
    float distance = 420, mileage = 15;
    float fuelNeeded = distance / mileage;
    printf("Fuel needed: %.2f liters\n", fuelNeeded);
    return 0;
