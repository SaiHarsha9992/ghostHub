Bucket Sort – Complete Guide

What is Bucket Sort?

Bucket Sort is a non-comparison-based, distribution-based sorting algorithm. It divides an array into several “buckets,” each holding a range of values. The values inside each bucket are then sorted using another algorithm (like Insertion Sort), and finally, the sorted buckets are concatenated to get the final sorted array.

Why Bucket Sort?

  • Efficient when the input is **uniformly distributed**.
  • Can run in linear time under ideal conditions: $O(n + k)$ (where $n$ is the number of elements and $k$ is the number of buckets).

How it works?

  1. Create an array of empty buckets.
  2. Scatter elements into buckets based on a mapping function (e.g., `floor(value * num_buckets)`).
  3. Sort each bucket individually.
  4. Concatenate all buckets to get the final sorted array.

Key Property: Bucket Sort is stable if the sorting algorithm used within the buckets is also stable.

Time Complexity

Case Time Complexity
Best $O(n + k)$
Average $O(n + n^2/k + k)$
Worst $O(n^2)$

Space complexity = $O(n + k)$. Note: If all elements fall into a single bucket, complexity can degrade to $O(n^2)$.

void insertionSort(float arr[], int n) {
    for (int i = 1; i < n; i++) {
        float key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

void bucketSort(float arr[], int n) {
    float buckets[n][n];
    int bucketSizes[n];
    for (int i = 0; i < n; i++) bucketSizes[i] = 0;

    for (int i = 0; i < n; i++) {
        int idx = n * arr[i];
        buckets[idx][bucketSizes[idx]++] = arr[i];
    }

    for (int i = 0; i < n; i++)
        insertionSort(buckets[i], bucketSizes[i]);

    int k = 0;
    for (int i = 0; i < n; i++)
        for (int j = 0; j < bucketSizes[i]; j++)
            arr[k++] = buckets[i][j];
}

int main() {
    float arr[] = {0.42, 0.32, 0.23, 0.52, 0.25, 0.47, 0.51};
    int n = sizeof(arr) / sizeof(arr[0]);
    bucketSort(arr, n);
    printf("Sorted array: ");
    for (int i = 0; i < n; i++) printf("%.2f ", arr[i]);
}

void bucketSort(vector& arr) {
    int n = arr.size();
    vector> buckets(n);

    for (float x : arr) {
        int idx = n * x;
        buckets[idx].push_back(x);

    for (int i = 0; i < n; i++)
        sort(buckets[i].begin(), buckets[i].end());

    int index = 0;
    for (int i = 0; i < n; i++)
        for (float x : buckets[i])
            arr[index++] = x;
}

Bucket Sort in Python

def bucket_sort(arr):
    n = len(arr)
    buckets = [[] for _ in range(n)]

    for x in arr:
        index = int(n * x)
        buckets[index].append(x)

    for bucket in buckets:
        bucket.sort()

    sorted_arr = []
    for bucket in buckets:
        sorted_arr.extend(bucket)

    return sorted_arr

arr = [0.42, 0.32, 0.23, 0.52, 0.25, 0.47, 0.51]
sorted_arr = bucket_sort(arr)
print("Sorted array:", sorted_arr)

Bucket Sort in Java

import java.util.*;

public class BucketSort {
    public static void bucketSort(float[] arr) {
        int n = arr.length;
        List[] buckets = new ArrayList[n];
        for (int i = 0; i < n; i++) buckets[i] = new ArrayList<>();

        for (float num : arr) {
            int idx = (int) (num * n);
            buckets[idx].add(num);
        }

        for (List bucket : buckets)
            Collections.sort(bucket);

        int index = 0;
        for (List bucket : buckets)
            for (float num : bucket)
                arr[index++] = num;
    }

    public static void main(String[] args) {
        float[] arr = {0.42f, 0.32f, 0.23f, 0.52f, 0.25f, 0.47f, 0.51f};
        bucketSort(arr);
        System.out.print("Sorted array: ");
        for (float num : arr) System.out.print(num + " ");
    }
}

When to Use Bucket Sort?

Best for:

  • Floating-point numbers between 0 and 1 (or values that can be normalized).
  • **Uniformly distributed** datasets.

Not suitable for:

  • Highly skewed data.
  • Huge datasets with unknown distribution (might lead to uneven buckets).

Final Thoughts

Bucket Sort is a highly intuitive, distribution-based sorting algorithm. Under the right conditions, it can achieve **linear time sorting**, outperforming traditional comparison-based sorts. It is frequently used in scientific computations, graphics, and numerical applications.
