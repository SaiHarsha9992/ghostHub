Strings and Numbers in JavaScript

1. What are Strings and Numbers?

  • String ‚Üí A sequence of characters (text), written inside quotes " " or ' '.
  • Number ‚Üí A numeric value (can be integer or decimal).

Example:

let name = "Durga";    // String
let age = 20;          // Number

2. Common String Methods

Strings in JavaScript have many built-in methods.

  • .length ‚Üí Returns the number of characters
  • .toUpperCase() ‚Üí Converts to uppercase
  • .toLowerCase() ‚Üí Converts to lowercase
  • .includes("word") ‚Üí Checks if a string contains a word
  • .slice(start, end) ‚Üí Extracts part of the string
  • .replace("old", "new") ‚Üí Replaces text
  • .trim() ‚Üí Removes spaces from start and end

Example:

let text = "  JavaScript is Fun!  ";

console.log(text.length);               // 21
console.log(text.toUpperCase());        // JAVASCRIPT IS FUN!
console.log(text.includes("Fun"));      // true
console.log(text.slice(0, 10));         // "JavaScript"
console.log(text.replace("Fun", "Powerful")); // "JavaScript is Powerful!"
console.log(text.trim());               // "JavaScript is Fun!"

3. Common Number Methods

Numbers also have useful methods.

  • .toFixed(n) ‚Üí Rounds a number to n decimal places
  • .toString() ‚Üí Converts a number to a string
  • parseInt("text") ‚Üí Converts a string to an integer
  • parseFloat("text") ‚Üí Converts a string to a decimal
  • Math.round() ‚Üí Rounds to the nearest integer
  • Math.floor() ‚Üí Rounds down
  • Math.ceil() ‚Üí Rounds up
  • Math.random() ‚Üí Returns a random number between 0 and 1

Example:

let num = 3.14159;
 
console.log(num.toFixed(2));      // 3.14
console.log(num.toString());      // "3.14159"
console.log(parseInt("100px"));     // 100
console.log(parseFloat("99.5kg"));  // 99.5
console.log(Math.round(4.7));     // 5
console.log(Math.floor(4.7));     // 4
console.log(Math.ceil(4.1));      // 5
console.log(Math.random());       // Random number like 0.5342

4. Real-Life Uses

  • Strings ‚Üí Used in names, messages, user input, and search bars
  • Numbers ‚Üí Used in calculations, scores, prices, and timers
