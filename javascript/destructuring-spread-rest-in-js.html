Destructuring, Spread, and Rest Operators

1. Introduction

JavaScript provides modern syntax features like **destructuring**, **spread**, and **rest** operators to make code cleaner, readable, and more efficient.

  • **Destructuring** ‚Üí Extract values from arrays or objects into variables.
  • **Spread (...)** ‚Üí Expand elements of an array or object.
  • **Rest (...)** ‚Üí Collect multiple elements into a single array or parameter.

These features are widely used in modern JavaScript and React development.

2. Destructuring

(A) Array Destructuring

let numbers = [10, 20, 30];
let [a, b, c] = numbers;

console.log(a); // 10
console.log(b); // 20
console.log(c); // 30

(B) Object Destructuring

let person = { name: "Durga", age: 20, city: "Hyderabad" };
let { name, age, city } = person;

console.log(name); // Durga
console.log(age);  // 20
console.log(city); // Hyderabad

(C) Default Values

let person = { name: "Durga" };
let { name, age = 18 } = person;

console.log(name); // Durga
console.log(age);  // 18 (default)

3. Spread Operator (...)

The spread operator expands arrays or objects.

(A) Copying Arrays

let arr1 = [1, 2, 3];
let arr2 = [...arr1];

console.log(arr2); // [1, 2, 3]

(B) Merging Arrays

let arr1 = [1, 2];
let arr2 = [3, 4];
let combined = [...arr1, ...arr2];

console.log(combined); // [1, 2, 3, 4]

(C) Spread in Objects

let person = { name: "Durga", age: 20 };
let updatedPerson = { ...person, city: "Hyderabad" };

console.log(updatedPerson); 
// { name: "Durga", age: 20, city: "Hyderabad" }

4. Rest Operator (...)

The rest operator collects remaining elements into a single array or parameter.

(A) Rest in Function Parameters

function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

console.log(sum(10, 20, 30)); // 60

(B) Rest in Destructuring

let numbers = [10, 20, 30, 40];
let [first, second, ...rest] = numbers;

console.log(first); // 10
console.log(second); // 20
console.log(rest); // [30, 40]

6. Advantages ‚úÖ

  • Makes code cleaner and readable
  • Avoids manual assignments and loops for copying/merging
  • Useful in React props, API data handling, and functional programming
  • Supports default values and dynamic arguments

7. Disadvantages ‚ùå

  • Can confuse beginners if overused
  • Shallow copies only with spread (nested objects/arrays require deep copy)
  • Misusing rest can lead to unexpected array structures
