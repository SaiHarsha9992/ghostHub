Variables and Data Types in C

In C you use variables to store information like numbers or characters. But C is different because you have to specify the type of data a variable will hold before you use it. This is like labeling your storage boxes so you know exactly what goes inside.

üì¶ What is a Variable in C?

A variable in C is a named memory location used to store data. You have to declare a variable by giving it a data type and a name before you can store a value in it.

Example:

int age = 20;

char initial = 'G';

Here, age is a variable storing the integer value 20, and initial is a variable storing the character 'G'.

üî¢ C Data Types

C has a variety of data types that define the kind of data a variable can hold and the amount of memory it will occupy.

1. Basic Data Types

  • int: Used for whole numbers (integers) without decimal points, like 10, -5, or 100.
  • float: Used for single-precision floating-point numbers with decimal points, like 3.14 or 0.99.
  • double: Used for double-precision floating-point numbers. They provide more precision than float, so they're used for more accurate calculations.
  • char: Used for single characters, like 'A', 'z', or '7'.

2. Derived Data Types

These are built from the basic data types.

  • Arrays: A collection of variables of the same type, like a list of ages.
  • Pointers: A special variable that stores the memory address of another variable. This is a powerful feature in C that allows for direct memory manipulation.

üåç Real-Life Examples of Variables

  • GPS app: It uses float or double variables to store latitude and longitude coordinates.
  • Text editor: It uses a char variable to store each character you type.
  • Calculator: It uses int or float variables to store the numbers you are performing calculations on.

üíª Example Code

#include 

int main() {
    // Declaring and initializing variables
    int student_id = 101;
    float average_score = 95.5;
    char grade = 'A';

    // Printing the values
    printf("Student ID: %d\n", student_id);
    printf("Average Score: %.2f\n", average_score);
    printf("Grade: %c\n", grade);

    return 0;
}

In this code, we declare variables with specific data types (int, float, char) and then use printf() with format specifiers (%d, %.2f, %c) to print their values.

‚úÖ Summary

  • Variables in C are labeled memory containers.
  • You must specify a data type when you declare a variable.
  • Data types like int, float, double, and char tell the compiler what kind of data the variable can hold.
  • C's strong typing helps the compiler manage memory efficiently, which is why C is so fast.
