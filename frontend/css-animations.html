CSS Animations

CSS Animations allow you to bring elements to life by smoothly changing their properties over time. They are used to create interactive and engaging webpages.

1. What is CSS Animation?

  • Animations change the style of an element gradually.
  • They are controlled using @keyframes and the animation property.
  • Animations can include moving, rotating, scaling, fading, or color changes.

Syntax:

selector {
  animation: name duration timing-function delay iteration-count direction fill-mode;
}

2. Keyframes

@keyframes define the stages of the animation.

Example:

@keyframes moveBox {
  0% { transform: translateX(0); }
  100% { transform: translateX(100px); }
}

3. Using the Animation Property

Example:

div {
  width: 50px;
  height: 50px;
  background-color: teal;
  animation: moveBox 2s linear infinite;
}
  • moveBox ‚Üí the @keyframes animation name
  • 2s ‚Üí duration
  • linear ‚Üí speed curve
  • infinite ‚Üí repeats forever

4. Common Animation Properties

  1. animation-name ‚Üí Name of the @keyframes.
  2. animation-duration ‚Üí How long it takes to complete one cycle.
  3. animation-timing-function ‚Üí Speed curve (linear, ease, ease-in, ease-out).
  4. animation-delay ‚Üí Wait time before starting animation.
  5. animation-iteration-count ‚Üí Number of times the animation runs (infinite for endless).
  6. animation-direction ‚Üí normal, reverse, alternate, etc.
  7. animation-fill-mode ‚Üí Keeps final state of animation (forwards, backwards).

6. Why We Need CSS Animations

  • Adds life and interactivity to webpages
  • Enhances user engagement and experience
  • Can highlight important content dynamically
  • Used for buttons, sliders, loaders, and cards

7. Advantages

  • Smooth visual effects without JavaScript
  • Lightweight and fast in modern browsers
  • Can be combined with transforms and transitions
  • Easy to loop, delay, or reverse animations

8. Conclusion

CSS Animations are powerful tools to make webpages interactive and dynamic. By mastering keyframes and the animation property, you can move, rotate, scale, and fade elements smoothly, creating visually appealing websites.
