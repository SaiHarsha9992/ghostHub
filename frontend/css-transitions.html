CSS Transitions

CSS Transitions allow you to smoothly change property values of an element over time, usually triggered by hover, focus, or other events. They are simpler than animations and are perfect for interactive effects.

1. What is a CSS Transition?

  • A transition animates the change between two states of an element.
  • It is event-triggered, like when a user hovers over a button.
  • Unlike animations, transitions do not loop automatically.

Syntax:

selector {
  transition: property duration timing-function delay;
}

2. Transition Properties

  • property ‚Üí The CSS property to animate (e.g., color, width, transform).
  • duration ‚Üí How long the transition takes (e.g., 0.5s, 2s).
  • timing-function ‚Üí Speed curve (linear, ease, ease-in, ease-out).
  • delay ‚Üí Time before the transition starts.

3. Small Examples

a) Change background color on hover

button {
  background-color: teal;
  color: white;
  transition: background-color 0.5s ease;
}

button:hover {
  background-color: orange;
}

‚úÖ The button background smoothly changes to orange when hovered.

b) Smooth size change

div {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  transition: width 0.5s, height 0.5s;
}

div:hover {
  width: 150px;
  height: 150px;
}

‚úÖ The box grows smoothly when hovered.

c) Transform example

div {
  transform: rotate(0deg);
  transition: transform 0.5s ease-in-out;
}

div:hover {
  transform: rotate(45deg);
}

‚úÖ The box rotates smoothly on hover.

5. Why We Need Transitions

  • Enhance user experience by adding smooth visual feedback
  • Make websites more interactive and engaging
  • Simple to implement compared to full animations
  • Useful for hover effects, buttons, cards, and menus

6. Advantages

  • Smooth visual effects with minimal code
  • Lightweight and fast
  • Works well with hover, focus, and active states
  • Can combine multiple properties at once

7. Conclusion

CSS Transitions are a simple yet powerful way to animate changes in element properties. By using duration, timing functions, and property changes, you can make your webpages interactive, dynamic, and visually appealing with minimal effort.
