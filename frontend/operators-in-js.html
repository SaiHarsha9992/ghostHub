Operators in JavaScript

1. What are Operators?

Operators are symbols that perform actions on values and variables. They help us calculate, compare, assign, and control logic in JavaScript.

Example:

let x = 10, y = 5;
console.log(x + y); // 15

2. Types of Operators in JavaScript

(A) Arithmetic Operators

Used for mathematical calculations.

  • + ‚Üí Addition
  • - ‚Üí Subtraction
  • * ‚Üí Multiplication
  • / ‚Üí Division
  • % ‚Üí Modulus (remainder)
  • ** ‚Üí Exponentiation (power)
  • ++ ‚Üí Increment (add 1)
  • -- ‚Üí Decrement (subtract 1)

Example:

let a = 10, b = 3;
console.log(a + b);   // 13
console.log(a % b);   // 1
console.log(a ** b);  // 1000
a++;
console.log(a);       // 11

(B) Assignment Operators

Used to assign values to variables.

  • = ‚Üí Assign
  • += ‚Üí Add and assign
  • -= ‚Üí Subtract and assign
  • *= ‚Üí Multiply and assign
  • /= ‚Üí Divide and assign
  • %= ‚Üí Modulus and assign

Example:

let x = 5;
x += 3;    // x = x + 3
console.log(x); // 8

(C) Comparison Operators

Used to compare values.

  • == ‚Üí Equal (ignores type)
  • === ‚Üí Strictly equal (checks type)
  • != ‚Üí Not equal
  • !== ‚Üí Strictly not equal
  • > ‚Üí Greater than
  • < ‚Üí Less than
  • >= ‚Üí Greater than or equal
  • <= ‚Üí Less than or equal

Example:

console.log(5 == "5");   // true
console.log(5 === "5");  // false
console.log(10 >= 7);    // true

(D) Logical Operators

Used for decision-making.

  • && ‚Üí AND (true if both are true)
  • || ‚Üí OR (true if one is true)
  • ! ‚Üí NOT (reverses the value)

Example:

let p = true, q = false;
console.log(p && q); // false
console.log(p || q); // true
console.log(!p);     // false

(E) Bitwise Operators

Work on numbers at binary level.

  • & ‚Üí AND
  • | ‚Üí OR
  • ^ ‚Üí XOR
  • ~ ‚Üí NOT
  • << ‚Üí Left shift
  • >> ‚Üí Right shift

Example:

console.log(5 & 1);  // 1
console.log(5 | 1);  // 5
console.log(5 ^ 1);  // 4
console.log(~5);     // -6
console.log(5 << 1); // 10

(F) Ternary (Conditional) Operator

A short way of writing if-else.

Example:

let age = 18;
let result = (age >= 18) ? "Adult" : "Minor";
console.log(result); // Adult

(G) Type Operators

Used to check the type of a value.

  • typeof ‚Üí Returns the data type
  • instanceof ‚Üí Checks if an object is an instance of a class

Example:

console.log(typeof "Hello");       // string
console.log(typeof 123);         // number
console.log([1,2,3] instanceof Array); // true
