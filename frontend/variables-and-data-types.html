Variables and Data Types in JavaScript

1. What are Variables?

A variable is like a container that stores data. In JavaScript, we use variables to store and reuse values.

Example:

let name = "Durga";
let age = 20;
console.log(name);
console.log(age);
  • Note: Here, name stores "Durga" and age stores 20.

2. Declaring Variables in JavaScript

There are three ways to declare variables:

  • var ‚Üí Old way (not recommended)
  • let ‚Üí Modern way, can be updated
  • const ‚Üí Fixed, cannot be changed

Example:

var city = "Hyderabad";   // Can be changed
let country = "India";     // Can be changed
const pi = 3.14;           // Cannot be changed

3. Data Types in JavaScript

JavaScript has two main types of data:

(A) Primitive Data Types

  • String ‚Üí Text values
  • Number ‚Üí Numbers (integer, float)
  • Boolean ‚Üí true/false
  • Undefined ‚Üí Not assigned a value
  • Null ‚Üí Empty value
  • Symbol ‚Üí Unique value
  • BigInt ‚Üí Large integers

(B) Non-Primitive Data Types

  • Objects
  • Arrays

Note: For now, let‚Äôs focus on String, Number, Boolean.

4. String (Text Values)

A string is written inside quotes " " or ' '.

Example:

let language = "JavaScript";
console.log(language.toUpperCase());  // JAVASCRIPT
console.log(language.length);         // 10

5. Number (Numeric Values)

Numbers can be integers or decimals.

Example:

let marks = 95;
let pi = 3.1416;
console.log(marks + 5);     // 100
console.log(pi.toFixed(2)); // 3.14

6. Boolean (True/False Values)

Booleans are used in conditions.

Example:

let isStudent = true;
let isTeacher = false;
console.log(isStudent);  // true
console.log(10 > 5);     // true
console.log(5 > 10);     // false

7. Advantages of Using Variables

  • Store values in memory
  • Reuse values anywhere in code
  • Easy to update and modify data
  • Makes code simple and readable
