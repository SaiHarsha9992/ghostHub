HTML Tables

1. What are Tables in HTML?

  • Tables are used to arrange data in rows and columns.
  • They are created using the <table> tag.
  • Each row is inside <tr> (table row).
  • Each header is inside <th> (table heading).
  • Each data cell is inside <td> (table data).

2. Basic Syntax

<table border="1">
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

‚úî border="1" adds a simple border.

4. Table with colspan and rowspan

  • colspan ‚Üí merge columns.
  • rowspan ‚Üí merge rows.

üëâ Example:
