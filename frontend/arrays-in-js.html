Arrays in JavaScript

1. Introduction

In JavaScript, arrays are used to store multiple values in a single variable. Arrays are ordered collections and can hold any type of data: numbers, strings, objects, or even other arrays.

Arrays are essential for managing lists of data efficiently and performing operations like looping, searching, and modifying elements.

2. How to Create Arrays

(A) Using Array Literals

let fruits = ["Apple", "Banana", "Mango"];
console.log(fruits); // ["Apple", "Banana", "Mango"]

(B) Using the Array Constructor

let numbers = new Array(1, 2, 3, 4);
console.log(numbers); // [1, 2, 3, 4]

(C) Empty Array and Adding Elements Later

let colors = [];
colors.push("Red");
colors.push("Blue");
console.log(colors); // ["Red", "Blue"]

3. Common Array Methods

(A) Adding & Removing Elements

let animals = ["Dog", "Cat"];

// Add at end
animals.push("Elephant");
console.log(animals); // ["Dog", "Cat", "Elephant"]

// Remove from end
animals.pop();
console.log(animals); // ["Dog", "Cat"]

// Add at beginning
animals.unshift("Lion");
console.log(animals); // ["Lion", "Dog", "Cat"]

// Remove from beginning
animals.shift();
console.log(animals); // ["Dog", "Cat"]

(B) Finding Elements

let numbers = [10, 20, 30, 40];
console.log(numbers.indexOf(20)); // 1
console.log(numbers.includes(30)); // true

(C) Looping Through Arrays

let fruits = ["Apple", "Banana", "Mango"];

// Using for loop
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// Using forEach
fruits.forEach(fruit => console.log(fruit));

(D) Modifying Elements

let colors = ["Red", "Blue", "Green"];
colors[1] = "Yellow";
console.log(colors); // ["Red", "Yellow", "Green"]

(E) Array Slicing and Splicing

let numbers = [10, 20, 30, 40, 50];

// slice(start, end) ‚Üí returns a new array
console.log(numbers.slice(1, 4)); // [20, 30, 40]

// splice(start, deleteCount, item1, item2...)
numbers.splice(2, 1, 35, 36); 
console.log(numbers); // [10, 20, 35, 36, 40, 50]

(F) Joining & Converting to String

let fruits = ["Apple", "Banana", "Mango"];
console.log(fruits.join(", ")); // "Apple, Banana, Mango"

(G) Array Concatenation

let arr1 = [1, 2];
let arr2 = [3, 4];
let combined = arr1.concat(arr2);
console.log(combined); // [1, 2, 3, 4]

(H) Sorting Arrays

let numbers = [30, 10, 20, 40];
numbers.sort((a, b) => a - b); // Ascending
console.log(numbers); // [10, 20, 30, 40]

5. Advantages of Arrays ‚úÖ

  • Store multiple values in a single variable.
  • Easy to loop through and perform operations.
  • Built-in methods make manipulation simple and efficient.
  • Can store mixed data types (numbers, strings, objects).

6. Disadvantages of Arrays ‚ùå

  • Fixed methods like push and pop can‚Äôt directly handle complex objects without extra logic.
  • Sparse arrays (arrays with empty slots) can cause unexpected behavior.
  • Large arrays may be memory-intensive.
