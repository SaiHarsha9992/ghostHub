HTML Elements, Comments, and Basic Tags

1. What are HTML Elements?

An HTML element is the building block of a webpage.

It usually has:

  • Opening Tag
  • Content
  • Closing Tag

üëâ Example:

<p>Hello World</p>

‚úî Together, these form an HTML Element.

2. What are HTML Tags?

  • Special keywords inside `< >`.
  • Most come in pairs (`<p> ... </p>`).
  • Some are self-closing (`<br>`, `<img>`).

3. What are Comments?

  • Notes for developers (not visible in browser).
  • Written with `<!-- -->`.

üëâ Example:

<!-- This is a comment -->
<p>Hello World</p>

4. Why Use Comments?

  • To explain code
  • To organize sections
  • To temporarily disable code
