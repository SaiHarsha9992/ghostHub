CSS Properties

CSS properties let you style your HTML elements. They define how elements look, where they appear, and how they behave.

1. What is a CSS Property?

A CSS property is the thing you want to change in an element, like color, size, spacing, or position.

Syntax:

selector {
  property: value;
}

Example:

p {
  color: blue;
  font-size: 18px;
}

Here:

  • p ‚Üí selector (which element to style)
  • color, font-size ‚Üí properties (what you want to change)
  • blue, 18px ‚Üí values (how you want it to look)

2. Common CSS Properties

  • Display
    Controls how an element is shown on the page.
  • Background
    Adds color or image behind an element.
  • Border
    Adds edges around elements.
  • Font
    Controls text style like font type, size, weight, and style.
  • Color
    Changes text color.
  • Margin
    Controls space outside an element.
  • Padding
    Controls space inside an element between content and border.
  • Position
    Controls where an element appears on the page.
  • White-Space
    Controls how text wraps inside an element.
  • Height and Width
    Defines size of an element.
  • Overflow
    Controls what happens when content is too big for its container.
  • Shadow
    Adds shadows to elements or text.

3. Why We Need CSS Properties

  • To style HTML content and make it visually appealing.
  • To control spacing and layout efficiently.
  • To improve readability and user experience.
  • To create modern and responsive designs.

4. Advantages of CSS Properties

  • Separate content and design ‚Üí cleaner HTML.
  • Reusable styles for multiple elements.
  • Easy updates and maintenance.
  • Makes website look professional and consistent.
