CSS Grid

CSS Grid is a powerful layout system that allows you to create two-dimensional layouts with rows and columns. It gives precise control over the placement and size of elements on a webpage.

1. What is CSS Grid?

  • CSS Grid divides a container into rows and columns.
  • Each item can be placed in a specific row and column.
  • Unlike Flexbox (1D layouts), Grid is perfect for complex 2D layouts.

2. Grid Container and Grid Items

  • Grid Container ‚Üí the parent element with display: grid
  • Grid Items ‚Üí direct children inside the container

Example:

.container {
  display: grid;
  grid-template-columns: 100px 100px 100px;
  grid-template-rows: 100px 100px;
  gap: 10px;
}

‚úÖ This creates a grid with 3 columns, 2 rows, and 10px gap between items.

3. Key Grid Properties

a) Grid Template Columns and Rows

Define the number and size of columns and rows.

.container {
  display: grid;
  grid-template-columns: 150px 150px 150px; /* 3 columns */
  grid-template-rows: 100px 100px;          /* 2 rows */
}

b) Gap

Controls spacing between grid items.

.container {
  gap: 15px; /* both row and column gaps */
}

c) Grid Column / Row for Items

Place items in specific columns or rows.

.item1 {
  grid-column: 1 / 3; /* spans from column 1 to 2 */
  grid-row: 1 / 2;    /* occupies row 1 */
}

d) Auto-Fill and Auto-Fit

Create dynamic grids that adjust based on screen size.

.container {
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
}

4. Small Example

.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-gap: 10px;
}

.item {
  background-color: lightcoral;
  height: 100px;
}

‚úÖ This creates 3 columns where the middle column is twice as wide, with 10px gaps.

6. Why We Need CSS Grid

  • Perfect for complex 2D layouts
  • Easier than floats or positioning for rows and columns
  • Responsive design becomes simpler
  • Precise control over placement and sizing of items

7. Conclusion

CSS Grid is a modern, powerful, and flexible layout system. By mastering properties like grid-template-columns, grid-template-rows, gap, and item placement, you can create structured, responsive, and visually appealing webpages easily.
