Introduction to HTML

       
           

● HTML stands for HyperText Markup Language.

           

● It is the foundation of every website on the internet.

           

● Think of it like the skeleton of a human body or the bricks of a house — it gives structure to a webpage.

       
       
           

👉 Example:

            
<h1>Hello World</h1>
<p>This is my first web page.</p>
       
       
           

2. Basic HTML Code Structure

           

Every HTML file follows this format:

            
<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>
       
                
           

3. Opening Tag, Closing Tag, and Content

           

HTML uses tags to mark elements.

           
               

👉 Example:

                
<p>Hello World</p>
           
           
                   
  • <p> → Opening Tag
    •                
  • Hello World → Content
    •                
  • </p> → Closing Tag
    •            
           

📌 Together, these form an HTML Element.

       
                
           

4. Basic HTML Tags

           
                   
  • Headings → <h1> to <h6>
    •                
  • Paragraphs → <p>
    •                
  • Links → <a href="https://google.com">Google</a>
    •                
  • Images → <img src="car.jpg" alt="Car">
    •                
  • Lists → <ul>, <ol>, <li>
    •            
       
       
           

5. Forms (User Input)

           
               

👉 Example:

                
<form>
 <label>Name:</label>
  <input type="text" placeholder="Enter your name">
  <button type="submit">Submit</button>
</form>
           
       
       
           

6. Real-Life Uses of HTML

           
               
                       
  • Login & signup forms
    •                    
  • Navigation bars
    •                    
  • Images and videos (YouTube, Instagram)
    •                    
  • Blogs, portfolios, resumes
    •                
           
       
       
           

7. Mini-Practice Tasks 📝

           
               
                       
  • Add your name as a heading.
    •                    
  • Write a short intro in a paragraph.
    •                    
  • Add an image of your favorite movie.
    •                    
  • Insert a link to your Instagram or GitHub.
    •                
           
       
       
           

8. Conclusion

           
                   
  • HTML is the first step in web development.
    •                
  • Opening Tag + Content + Closing Tag = Element.
    •                
  • Once you master HTML, you can build your own static webpages.
    •
