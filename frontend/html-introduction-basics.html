2. Basic HTML Code Structure

Every HTML file follows this format:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

  • Links ‚Üí <a href="https://google.com">Google</a>
