CSS Display and Positioning

CSS display and position properties help control how elements appear and where they are placed on a webpage.

1. CSS Display

Block: Takes full width, starts on a new line.

div {
  display: block;
}

Inline: Takes only the width needed, stays on the same line.

span {
  display: inline;
}

Inline-Block: Like inline but can set width/height.

button {
  display: inline-block;
  width: 100px;
}

None: Hides the element.

p {
  display: none;
}

Flex: Align child elements easily.

nav {
  display: flex;
  justify-content: space-between;
}

2. CSS Positioning

Static (default)

div {
  position: static; /* normal flow */
}

Relative (moves relative to original position)

div {
  position: relative;
  top: 10px;  /* moves down 10px */
  left: 20px; /* moves right 20px */
}

Absolute (moves relative to nearest positioned ancestor)

div {
  position: absolute;
  top: 50px;
  left: 100px;
}

Fixed (stays on screen even when scrolling)

header {
  position: fixed;
  top: 0;
  width: 100%;
  background-color: lightblue;
}

Sticky (behaves like relative until scrolling, then fixed)

h2 {
  position: sticky;
  top: 0; /* sticks to top when scrolling */
  background-color: yellow;
}
