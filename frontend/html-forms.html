HTML Forms

1. What are Forms in HTML?

  • Forms are used to collect user input on a webpage.
  • Created using the <form> tag.
  • Common inputs: text box, password, radio buttons, checkboxes, dropdowns, buttons.

2. Basic Syntax

<form action="submit.php" method="post">
  <!-- input fields here -->
</form>

‚úî action ‚Üí where the form data is sent.

‚úî method ‚Üí how data is sent (get or post).

6. Conclusion

  • Forms collect user input using <form>.
  • Inputs: text, password, radio, checkbox, select, textarea, buttons.
  • Attributes: action (where to send), method (how to send).
  • Forms are the backbone of user interaction on websites.
