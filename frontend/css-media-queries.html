CSS Media Queries

Media Queries are a powerful CSS feature that allows you to create responsive designs. They let your webpage adapt to different screen sizes like mobiles, tablets, or desktops.

1. What are Media Queries?

  • Media Queries apply different styles depending on the device‚Äôs characteristics, mainly the width, height, orientation, or resolution.
  • They are essential for responsive web design.

Syntax:

@media (condition) {
  /* CSS rules */
}

2. Common Media Query Conditions

  • max-width ‚Üí apply styles when screen is smaller than specified width.
  • min-width ‚Üí apply styles when screen is larger than specified width.
  • orientation ‚Üí apply styles for portrait or landscape mode.
  • max-height / min-height ‚Üí based on height of the screen.

3. Small Examples

a) Change background on small screens

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

‚úÖ Background changes when the screen width is 600px or smaller.

b) Adjust text size for larger screens

  p {
    font-size: 20px;
  }
}

‚úÖ Paragraph text becomes larger on big screens.

c) Change layout for tablets

  .container {
    flex-direction: column;
  }
}

‚úÖ Flex container switches to column layout for tablets and smaller devices.

5. Why We Need Media Queries

  • Create mobile-friendly and responsive websites.
  • Improve user experience across devices.
  • Avoid layout issues on smaller or larger screens.
  • Essential for modern web design standards.

6. Advantages

  • Makes your site look good on all devices
  • Reduces scrolling and zooming for mobile users
  • Helps maintain a consistent design
  • Supports flexible and adaptive layouts

7. Conclusion

Media Queries are key for responsive design. By using conditions like max-width, min-width, and orientation, you can apply different styles for different devices, ensuring your website is accessible, readable, and visually appealing everywhere.
