1. Introduction to HTML

- HTML stands for **HyperText Markup Language**.

- It is the foundation of every website you see on the internet.

- Think of it like the skeleton of a body or the bricks of a house ‚Äî it gives structure to a webpage.

- Without HTML, there would be no headings, paragraphs, images, or links.

2. Basic HTML Code Structure

Every HTML file follows this simple structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

**Explanation:**

  • <!DOCTYPE html> ‚Üí tells the browser this is an HTML5 document.
  • <html> ‚Üí the root of the page (everything goes inside it).
  • <head> ‚Üí contains info about the page (title, metadata).
  • <body> ‚Üí contains what you actually see on the webpage.

3. Opening Tag, Closing Tag, and Content

HTML uses tags to mark elements.

**Example:**

<p>Hello World</p>
  • <p> ‚Üí Opening Tag
  • Hello World ‚Üí Content
  • </p> ‚Üí Closing Tag

üìå Together, these form an **HTML Element**.

  • **Note:** Some tags don‚Äôt need closing (like <img> or <br>). They are called **self-closing tags**.

4. Common HTML Tags

  • **Headings** ‚Üí <h1> to <h6>
  • **Paragraph** ‚Üí <p>
  • **Links** ‚Üí <a href="https://google.com">Visit Google</a>
  • **Images** ‚Üí <img src="car.jpg" alt="Car">
  • **Lists** ‚Üí 
    <ul>
  <li>Apple</li>
  <li>Mango</li>
</ul>
