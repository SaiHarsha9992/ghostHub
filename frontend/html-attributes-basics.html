HTML Attributes ‚Äì Adding Extra Power to Tags

1. What is an Attribute?

In HTML, a tag alone gives structure, but an attribute gives extra information about that tag.

Think of attributes as **adjectives** for HTML tags ‚Äî they describe or modify the element.

Attributes always appear inside the opening tag.

üëâ Example:

<a href="https://google.com">Visit Google</a>

Here,

  • <a> is the tag (anchor tag for links).
  • href="https://google.com" is the attribute.
  • It tells the browser where the link should go when clicked.

2. Why Do We Use Attributes?

Without attributes, HTML is plain and basic. With attributes, we can:

  • Add links to text or images (`href`)
  • Show images or videos (`src`)
  • Add alternative text for accessibility (`alt`)
  • Provide extra meaning like tooltips (`title`)
  • Control size and style (`width`, `height`, `style`)
  • Identify or group elements (`id`, `class`)

üìå Example without attribute: `<img>`

üëâ This shows nothing because the browser doesn‚Äôt know which image.

üìå Example with attribute:

<img src="car.jpg" alt="Red Car">

üëâ Now it shows the image and gives text if the image fails to load.

3. Components of an Attribute

Every attribute has 3 parts:

  1. Attribute Name ‚Üí what property you want to set (like `href`, `src`, `id`)
  2. Equal Sign (=) ‚Üí connects name and value
  3. Attribute Value ‚Üí the actual setting, always inside quotes

üëâ Example:

<a href="https://example.com">Click Me</a>
  • `href` ‚Üí Attribute Name
  • `=` ‚Üí Equal Sign
  • `"https://example.com"` ‚Üí Attribute Value

4. Types of Attributes

üîπ Global Attributes

  • Can be used with any HTML tag.
  • Examples:
    • `id` ‚Üí gives a unique name to an element
    • `class` ‚Üí groups multiple elements together
    • `style` ‚Üí inline CSS for quick styling
    • `title` ‚Üí shows a tooltip on hover

üëâ Example:

<p id="intro" class="highlight" title="This is a tooltip">Hello World</p>

üîπ Specific Attributes

  • Work only with particular tags.
  • Examples:
    • `href` (only for `<a>`)
    • `src`, `alt`, `width`, `height` (for `<img>`)
    • `rows`, `cols` (for `<textarea>`)

üëâ Example:

<img src="flower.png" alt="Beautiful Flower" width="200" height="150">

5. Commonly Used Attributes with Examples

Example 1: Link with `href`

<a href="https://github.com">My GitHub</a>

‚úî Clicking the text opens GitHub in the browser.

Example 2: Image with `src`, `alt`, and size

<img src="car.jpg" alt="Sports Car" width="400" height="250">

‚úî Shows an image

‚úî If the image is missing, the text ‚ÄúSports Car‚Äù will appear

‚úî The image is resized to 400px √ó 250px

Example 3: Tooltip with `title`

<p title="I am a tooltip">Hover over me!</p>

‚úî When you hover the mouse, a small tooltip appears.

Example 4: Using `id` and `class`

<h1 id="main-title">Welcome to My Website</h1>
<p class="highlight">This is an important note.</p>
<p class="highlight">This is another important note.</p>

‚úî `id="main-title"` uniquely identifies the heading

‚úî `class="highlight"` groups multiple paragraphs together

Example 5: Styling with `style`

<p style="color: blue; font-size: 20px;">Styled Text</p>

‚úî Changes the color and size of the text directly in HTML

6. Rules for Using Attributes

  • ‚úÖ Always written inside the **opening tag**
  • ‚úÖ Values should be inside **double quotes `"`**
  • ‚úÖ You can use **multiple attributes** in the same tag

üëâ Example:

<img src="car.png" alt="Red Car" width="300" height="200">

8. Conclusion

  • Attributes are like extra details for HTML tags.
  • Every attribute has a name + value.
  • Two types: **Global** (work everywhere) and **Specific** (tag-based).
  • Common attributes: `href`, `src`, `alt`, `id`, `class`, `style`, `title`, `width`, `height`.
  • With attributes, your web pages become more useful, interactive, and accessible.
