HTML Multimedia

1. What is Multimedia in HTML?

  • Multimedia means using sound, video, or external content in webpages.
  • HTML provides tags like <audio>, <video>, and <iframe> for multimedia.

2. Audio in HTML

  • The <audio> tag is used to play sound/music.
  • Common attributes:
    • controls ‚Üí shows play/pause buttons.
    • autoplay ‚Üí starts playing automatically.
    • loop ‚Üí repeats audio.

3. Video in HTML

  • The <video> tag is used to play videos.
  • Common attributes:
    • controls ‚Üí play/pause buttons.
    • autoplay ‚Üí start automatically.
    • loop ‚Üí repeat video.
    • width & height ‚Üí size of video.

üëâ Example:

Video Example

<video width="400" height="250" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

4. Iframe in HTML

  • The <iframe> tag is used to embed another webpage inside the current page.

üëâ Example:

Iframe Example

5. Full Example
