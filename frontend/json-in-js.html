JSON in JavaScript

1. Introduction

JSON (JavaScript Object Notation) is a lightweight data format used for storing and exchanging data. It is widely used in APIs, server communication, and web applications.

In JavaScript, we often convert JSON to objects to manipulate data and convert objects to JSON for sending data to a server.

2. JSON Syntax Rules

  • Data is in key-value pairs.
  • Keys must be strings enclosed in double quotes.
  • Values can be strings, numbers, objects, arrays, true, false, or null.
  • Arrays are enclosed in [ ], objects in { }.

Example JSON:

{
  "name": "Durga",
  "age": 20,
  "city": "Hyderabad",
  "hobbies": ["singing", "reading"]
}

3. Parsing JSON (Converting JSON to Object)

Use JSON.parse() to convert a JSON string into a JavaScript object.

let jsonData = '{"name":"Durga","age":20,"city":"Hyderabad"}';
let person = JSON.parse(jsonData);

console.log(person.name); // Durga
console.log(person.age);  // 20

‚óè Tip: Ensure the JSON string is properly formatted, otherwise JSON.parse() will throw an error.

4. Stringifying Objects (Converting Object to JSON)

Use JSON.stringify() to convert a JavaScript object into a JSON string.

let person = {
    name: "Durga",
    age: 20,
    city: "Hyderabad"
};

let jsonString = JSON.stringify(person);
console.log(jsonString);
// {"name":"Durga","age":20,"city":"Hyderabad"}

‚óè Useful for sending data to a server or storing in local storage.

5. Formatting JSON

JSON.stringify can take additional parameters to format JSON nicely.

let person = {
    name: "Durga",
    age: 20,
    city: "Hyderabad"
};

let formattedJSON = JSON.stringify(person, null, 4);
console.log(formattedJSON);
/*
{
    "name": "Durga",
    "age": 20,
    "city": "Hyderabad"
}
*/

‚óè The third parameter specifies the number of spaces for indentation.

7. Advantages of JSON ‚úÖ

  • Lightweight and easy to read/write
  • Standard format for data exchange between client and server
  • Works with almost all programming languages
  • Can represent complex data structures like arrays and nested objects

8. Disadvantages of JSON ‚ùå

  • Keys must always be strings in double quotes
  • Cannot store functions, undefined, or special JavaScript objects like Date directly
  • Large JSON files may be slow to parse in the browser
