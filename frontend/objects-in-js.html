Objects and Object Methods

1. Introduction

In JavaScript, objects are used to store key-value pairs. They allow you to group related data and functionality together.

Objects are essential for representing real-world entities, such as users, products, or settings, and can include methods (functions inside objects) to perform actions on that data.

2. How to Create Objects

(A) Using Object Literals

let person = {
    name: "Durga",
    age: 20,
    city: "Hyderabad"
};

console.log(person.name); // Durga
console.log(person["age"]); // 20

(B) Using the new Object() Syntax

let car = new Object();
car.brand = "Toyota";
car.model = "Corolla";
console.log(car.brand); // Toyota

(C) Using a Constructor Function

function Person(name, age) {
    this.name = name;
    this.age = age;
}

let durga = new Person("Durga", 20);
console.log(durga.name); // Durga

3. Adding, Updating, and Deleting Properties

let book = {
    title: "JavaScript Guide",
    author: "MDN"
};

// Add
book.year = 2023;

// Update
book.title = "JS Complete Guide";

// Delete
delete book.author;

console.log(book); // { title: "JS Complete Guide", year: 2023 }

4. Object Methods

Methods are functions inside objects that can manipulate the object‚Äôs data.

(A) Defining Methods

let user = {
    name: "Durga",
    age: 20,
    greet: function() {
        console.log("Hello, " + this.name + "!");
    }
};

user.greet(); // Hello, Durga!

(B) Shorthand Method Syntax

let user = {
    name: "Durga",
    age: 20,
    greet() {
        console.log("Hello, " + this.name + "!");
    }
};

user.greet(); // Hello, Durga!

(C) Using this Keyword

this refers to the current object inside a method.

let car = {
    brand: "Toyota",
    showBrand() {
        console.log("Car brand is " + this.brand);
    }
};

car.showBrand(); // Car brand is Toyota

5. Common Object Methods

let person = {
    name: "Durga",
    age: 20,
    city: "Hyderabad"
};

// Object.keys() ‚Üí Returns array of keys
console.log(Object.keys(person)); // ["name", "age", "city"]

// Object.values() ‚Üí Returns array of values
console.log(Object.values(person)); // ["Durga", 20, "Hyderabad"]

// Object.entries() ‚Üí Returns array of [key, value] pairs
console.log(Object.entries(person)); 
// [["name","Durga"], ["age",20], ["city","Hyderabad"]]

// Object.assign() ‚Üí Copy properties from one object to another
let newPerson = {};
Object.assign(newPerson, person);
console.log(newPerson); // { name: "Durga", age: 20, city: "Hyderabad" }

7. Advantages of Objects ‚úÖ

  • Store related data together.
  • Can include functions to operate on the data.
  • Make code modular and organized.
  • Can be easily copied and manipulated with built-in methods.

8. Disadvantages of Objects ‚ùå

  • Objects are unordered, so key order may not be guaranteed in some operations.
  • Deep copying can be tricky; Object.assign() only performs shallow copies.
  • Overusing objects for simple data may be unnecessary overhead.
