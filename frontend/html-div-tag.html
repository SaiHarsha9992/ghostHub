HTML `<div>` Tag

1. What is `<div>`?

‚óè `<div>` stands for division.

‚óè It is a block-level container used to group elements together.

‚óè Helps in structuring webpages into sections (like header, content, footer).

üëâ Example:

<div>
  <h2>My Section</h2>
  <p>This is a paragraph inside a div.</p>
</div>

2. Why Use `<div>`?

‚úî Groups related content.

‚úî Makes webpage layout organized.

‚úî Can hold headings, paragraphs, images, tables, forms, etc.

‚úî Often used before applying styles or scripts (though here we only focus on HTML).

3. Div as a Container

<div>
  <h2>About Me</h2>
  <p>My name is Durga. I love coding and singing.</p>
</div>

<div>
  <h2>My Hobbies</h2>
  <ul>
    <li>Reading Books</li>
    <li>Music</li>
    <li>Coding</li>
  </ul>
</div>

‚úî Each `<div>` acts like a separate block.

4. Nested `<div>`

‚óè You can put one `<div>` inside another.

<div>
  <h2>Profile</h2>
  <div>
    <p>Name: Durga</p>
    <p>College: AEC</p>
  </div>
</div>

5. Real-Time Use of `<div>`

Think of a newspaper:

  • It has sections like Headlines, Sports, Entertainment.
  • In a webpage, `<div>` is used to separate these parts.

6. Full Example

<!DOCTYPE html>
<html>
<head>
  <title>Div Example</title>
</head>
<body>
  <h1>My Website</h1>
  <div>
    <h2>Header Section</h2>
    <p>Welcome to my site!</p>
  </div>
  <div>
    <h2>Main Content</h2>
    <p>Here you will find my blogs and tutorials.</p>
  </div>
  <div>
    <h2>Footer</h2>
    <p>© 2025 My Website</p>
  </div>
</body>
</html>

8. Conclusion

  • `<div>` is a block-level container.
  • Used to group and organize webpage content.
  • Can contain any HTML elements.
  • Acts like a section divider for webpages.
