Functions and Their Types in JavaScript

1. Introduction

A function is a block of code designed to perform a specific task.

Instead of writing the same code again and again, we wrap it inside a function and call it whenever needed.

2. Function Declaration

The most common way of creating functions.

üëâ Example:

  console.log("Hello, " + name + "!");

3. Function Expression

Functions can be stored inside variables.

üëâ Example:

const add = function(a, b) {
  return a + b;
};

4. Arrow Functions (ES6)

Introduced in ES6, a shorter way to write functions.

üëâ Example:

console.log(multiply(4, 5)); // 20

5. Anonymous Functions

Functions without a name. Often used in function expressions or callbacks.

üëâ Example:

setTimeout(function() {
  console.log("This runs after 2 seconds");
}, 2000);

6. Named Functions

Functions that have a name (opposite of anonymous).

üëâ Example:

function sayHello() {
  console.log("Hello!");
}
sayHello();

7. Immediately Invoked Function Expression (IIFE)

Functions that run immediately after creation.

üëâ Example:

(function() {
  console.log("I run automatically!");
})();

8. Callback Functions

A function passed as an argument to another function.

üëâ Example:

function processUserInput(callback) {
  let name = "Durga";
  callback(name);
}

processUserInput(function(userName) {
  console.log("Hello, " + userName);
});
