Advanced HTML Input Types & Attributes

1. What are Advanced Input Types?

  • HTML5 introduced new input types to make forms more powerful.
  • These inputs help collect data like emails, dates, numbers, colors, etc.

3. Important Input Attributes

üîπ required

Makes a field mandatory.

<input type="text" name="username" required>

üîπ placeholder

Shows a hint inside the input box.

<input type="text" name="city" placeholder="Enter your city">

üîπ readonly

Makes input uneditable.

<input type="text" value="India" readonly>

üîπ disabled

Disables the input field.

<input type="text" value="Not Editable" disabled>

üîπ maxlength

Limits characters in input.

<input type="text" name="pin" maxlength="6">

üîπ pattern

Ensures the input follows a specific rule (regular expression).

<input type="text" name="pin" pattern="[0-9]{6}" placeholder="6-digit PIN">
