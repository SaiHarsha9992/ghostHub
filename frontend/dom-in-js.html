Introduction to the DOM

1. Introduction

The **DOM (Document Object Model)** is a programming interface for web pages. It represents the page as a tree of objects (nodes), allowing JavaScript to access, modify, and manipulate HTML and CSS dynamically. Understanding the DOM is essential for making interactive web pages.

2. DOM Structure

  • Every HTML page is a document object.
  • Elements like <div>, <p>, and <button> are nodes in the DOM tree.
  • JavaScript can traverse, access, and modify these nodes.

Example:

<!DOCTYPE html>
<html>
<body>
  <h1 id="heading">Hello, World!</h1>
  <p class="text">This is a paragraph.</p>
</body>
</html>
  • The <h1> and <p> tags are nodes.
  • document is the root object representing the page.

3. Selecting Elements in the DOM

(A) By ID

let heading = document.getElementById("heading");
console.log(heading.textContent); // Hello, World!

(B) By Class Name

let paragraphs = document.getElementsByClassName("text");
console.log(paragraphs[0].textContent); // This is a paragraph.

(C) By Tag Name

let allParagraphs = document.getElementsByTagName("p");
console.log(allParagraphs.length); // 1

(D) Using querySelector

‚óè Selects the first matching element using a CSS selector

let firstParagraph = document.querySelector(".text");
console.log(firstParagraph.textContent); // This is a paragraph.

(E) Using querySelectorAll

‚óè Selects all matching elements using a CSS selector

let allParagraphs = document.querySelectorAll("p");
allParagraphs.forEach(p => console.log(p.textContent));

5. Advantages ‚úÖ

  • Makes web pages interactive and dynamic
  • Allows modifying content, styles, and attributes in real time
  • Can respond to user events like clicks, typing, and scrolling
  • Works with all modern browsers

6. Disadvantages ‚ùå

  • Direct DOM manipulation can be slow for large documents
  • Excessive use of DOM methods may make code hard to maintain
  • Beginners may get confused with node lists vs arrays

7. How to Save and Run DOM Examples

1. Open a text editor (VS Code, Sublime, Notepad++).

2. Write your HTML and JavaScript code.

3. Save the file with a .html extension, e.g., dom.html.

4. Open it in a browser to see the results.

Example HTML:

<!DOCTYPE html>
<html>
<head>
  <title>DOM Example</title>
</head>
<body>
  <h1 id="heading">Hello, World!</h1>
  <p class="text">This is a paragraph.</p>
  
  <script>
    let heading = document.getElementById("heading");
    heading.textContent = "Welcome to the DOM!";
  </script>
</body>
</html>

8. Conclusion

  • The DOM represents HTML documents as objects, allowing dynamic access and modification.
  • Use getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll to select elements.
  • Mastering DOM selection is essential for interactive web pages and event handling.

üìå Tip: Start by selecting elements and changing their text or style before moving to event handling and DOM manipulation.
