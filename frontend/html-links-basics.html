HTML Links ‚Äì Connecting Web Pages

1. What are Links in HTML?

‚óè Links (also called **Hyperlinks**) are used to connect one webpage to another.

‚óè They are created using the **<a> (anchor) tag**.

‚óè Clicking a link will take the user to another page, section, or even a file.

üëâ Example:

<a href="https://google.com">Visit Google</a>

2. Syntax

<a href="URL">Link Text</a>

‚úî `**href**` (Hypertext Reference) is the attribute that stores the destination URL.

‚úî The text inside `<a> ... </a>` is what the user clicks.

3. Types of Links

üîπ 1. Absolute Link

‚óè Full web address.

<a href="https://www.wikipedia.org">Wikipedia</a>

üîπ 2. Relative Link

‚óè Used to link to pages within the same project.

üîπ 3. Email Link

<a href="mailto:example@gmail.com">Send Email</a>

üîπ 4. Telephone Link

4. Opening Links in New Tab

Use the `target="_blank"` attribute.

<a href="https://openai.com" target="_blank">OpenAI Website</a>

5. Adding a Tooltip to a Link

Use the `title` attribute to show extra information on hover.

<a href="https://github.com" title="Visit GitHub">GitHub</a>

7. Conclusion

  • Links are made with the `<a>` tag.
  • `href` defines the destination.
  • `target="_blank"` opens links in a new tab.
  • Links can connect websites, project pages, emails, or phone numbers.
