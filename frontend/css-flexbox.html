CSS Flexbox

Flexbox is a modern CSS layout system that helps you align and distribute space among elements in a container, even when their size is unknown or dynamic.

1. What is Flexbox?

Flexbox (Flexible Box) makes it easier to:

  • Align items horizontally or vertically
  • Distribute space evenly
  • Create responsive layouts without complex calculations

Think of a flex container as a row or column where items can grow, shrink, and align automatically.

2. Flex Container and Flex Items

  • Flex Container ‚Üí the parent element with display: flex
  • Flex Items ‚Üí the direct children of the flex container

Example:

.container {
  display: flex;
}

All direct children of .container become flex items.

3. Main Flexbox Properties

a) Flex Direction

Defines the direction of items inside the container.

.container {
  display: flex;
  flex-direction: row; /* row, row-reverse, column, column-reverse */
}

b) Justify Content

Aligns items horizontally along the main axis.

.container {
  display: flex;
  justify-content: space-between; /* flex-start, flex-end, center, space-between, space-around */
}

c) Align Items

Aligns items vertically along the cross axis.

.container {
  display: flex;
  align-items: center; /* flex-start, flex-end, center, stretch, baseline */
}

d) Flex Wrap

Controls whether items wrap to the next line.

.container {
  display: flex;
  flex-wrap: wrap; /* nowrap, wrap, wrap-reverse */
}

e) Flex Grow, Flex Shrink, Flex Basis

Control how items grow or shrink relative to others.

.item {
  flex: 1; /* grow to fill available space */
}

4. Small Example

.container {
  display: flex;
  justify-content: space-around;
  align-items: center;
  height: 100px;
  background-color: lightgray;
}

.item {
  width: 50px;
  height: 50px;
  background-color: teal;
}

‚úÖ Three boxes inside .container will align horizontally, spaced evenly, and centered vertically.

6. Why We Need Flexbox

  • Easier horizontal and vertical alignment
  • Helps build responsive designs quickly
  • Reduces complex CSS with floats and positioning
  • Automatically adjusts spacing and size of items

7. Conclusion

Flexbox is a powerful and modern layout tool in CSS. Mastering properties like flex-direction, justify-content, align-items, flex-wrap, and flex allows you to create clean, responsive, and well-aligned layouts easily.
