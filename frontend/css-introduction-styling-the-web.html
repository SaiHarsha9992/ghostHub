1. Introduction to CSS

  • CSS stands for Cascading Style Sheets.
  • If HTML is the skeleton of a webpage, CSS is the skin, clothes, and design that make it look beautiful.
  • With CSS, you can change colors, fonts, spacing, layouts, and even add animations.
  • Without CSS, all websites would look plain like a Word document!

2. Why Do We Need CSS?

  • Separates content (HTML) from design (CSS).
  • Makes websites look attractive and user-friendly.
  • Helps websites work on different screen sizes (mobile, tablet, desktop).
  • Allows you to reuse styles across multiple pages.

4. Basic CSS Syntax

A CSS rule looks like this:

selector {
  property: value;
}

Example:

h1 {
  color: purple;
  font-size: 30px;
}
  • h1 ‚Üí Selector (which element to style)
  • color, font-size ‚Üí Properties
  • purple, 30px ‚Üí Values

5. Common CSS Properties

  • Colors ‚Üí color: red; background-color: yellow;
  • Text ‚Üí font-size, font-family, text-align, font-weight
  • Box Model ‚Üí margin, padding, border, width, height
  • Layout ‚Üí display, position, flexbox, grid

7. Conclusion

  • HTML = Structure, CSS = Style.
  • CSS makes websites colorful, modern, and responsive.
  • Start with simple properties like colors and fonts, then move to layouts and animations.
  • Mastering CSS is the first step to becoming a web designer or front-end developer.
