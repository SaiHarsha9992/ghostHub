HTML Symbols

1. What are HTML Symbols?

‚óè HTML provides many special symbols (mathematical, currency, arrows, etc.) using entities.

‚óè These symbols are not typed directly on the keyboard.

‚óè They are written as `&name;` or `&#number;`.

2. Common Symbol Categories

üîπ Mathematical Symbols

<p>10 × 5 = 50</p> <!-- Multiplication -->
<p>20 ÷ 4 = 5</p> <!-- Division -->
<p>5 + 3 = 8</p>   <!-- Plus -->
<p>9 − 4 = 5</p>   <!-- Minus -->
<p>10 ≤ 20</p>   <!-- Less than or equal -->
<p>15 ≥ 10</p>   <!-- Greater than or equal -->

üîπ Currency Symbols

<p>Dollar: &#36;100</p>
<p>Rupee: &#8377;500</p>
<p>Euro: &euro;50</p>
<p>Pound: &pound;40</p>
<p>Yen: &yen;2000</p>

üîπ Arrow Symbols

<p>Left Arrow: &larr;</p>
<p>Right Arrow: &rarr;</p>
<p>Up Arrow: &uarr;</p>
<p>Down Arrow: &darr;</p>
<p>Double Arrow: &harr;</p>

üîπ Other Useful Symbols

<p>Copyright: &copy; 2025</p>
<p>Trademark: &trade;</p>
<p>Registered: &reg;</p>
<p>Heart: &hearts;</p>
<p>Star: &#9733;</p>
<p>Checkmark: &#10004;</p>

5. Conclusion

  • HTML symbols are written using entities.
  • Categories: Math, Currency, Arrows, Special Characters.
  • Format: `&name;` or `&#number;`.
