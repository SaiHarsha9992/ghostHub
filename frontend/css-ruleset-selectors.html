CSS Ruleset and Selectors

1. What is a CSS Ruleset?

A ruleset in CSS tells the browser which element to style and how to style it.

It has 2 main parts:

selector {
  property: value;
}

üëâ Example:

  color: blue;
  font-size: 18px;
}
  • p ‚Üí Selector (which element to target)
  • color, font-size ‚Üí Properties (what you want to change)
  • blue, 18px ‚Üí Values (the actual style applied)

So, this ruleset says: ‚ÄúMake all paragraphs blue and 18px in size.‚Äù

2. What is a Selector?

A selector chooses which HTML element(s) you want to style. Think of it as ‚Äúwho‚Äù you are talking to in your CSS.

üîπ Universal Selector (*) ‚Äì selects everything.

üîπ Type Selector (Element Selector) ‚Äì selects all elements of a type.

4. Example: Without Selector vs With Selector

Without Selector (no CSS):

<p>Hello World</p>
<h1>Welcome!</h1>

With CSS Selectors:

<style>
  p {
    color: blue;
  }
  h1 {
    color: darkred;
    text-align: center;
  }
</style>
<p>Hello World</p>
<h1>Welcome!</h1>

üëÄ Output:

  • Paragraph turns blue.
  • Heading turns dark red and is centered.

6. Conclusion

  • A ruleset = selector + property + value.
  • Selectors decide which elements to style.
  • By mastering selectors, you can control every part of your webpage design.
