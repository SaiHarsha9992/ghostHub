HTML Images

1. What are Images in HTML?

‚óè Images are added to webpages using the `<img>` tag.

‚óè The `<img>` tag is self-closing (no closing tag).

‚óè Images make a webpage more attractive and informative.

2. Syntax

<img src="image.jpg" alt="Description">

‚úî `src` ‚Üí specifies the image file path or URL.

‚úî `alt` ‚Üí provides alternate text (shown if the image is missing).

3. Example of Adding an Image

<!DOCTYPE html>
<html>
<head>
  <title>Image Example</title>
</head>
<body>
  <h2>My Favorite Car</h2>
  <img src="car.jpg" alt="A red sports car">
</body>
</html>

4. Image with Width and Height

You can change the size of an image with the `width` and `height` attributes.

<img src="flower.jpg" alt="Flower" width="300" height="200">

5. Image from a Website (Absolute URL)

You can also display an image directly from the internet.

<img src="https://www.w3schools.com/html/img_chania.jpg" alt="Sea View">

6. Image as a Link

You can make an image clickable.

<a href="https://google.com">
  <img src="google.png" alt="Google Logo" width="150">
</a>

8. Conclusion

  • `<img>` is used to insert images.
  • `src` defines the source of the image.
  • `alt` provides alternate text for accessibility.
  • `width` and `height` change the image size.
  • Images can also act as clickable links.
