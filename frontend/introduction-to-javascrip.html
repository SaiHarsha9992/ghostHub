3. How to Use JavaScript in a Webpage

There are 3 ways to add JavaScript:

Inline JavaScript (inside HTML tag)

<button onclick="alert('Hello!')">Click Me</button>

Internal JavaScript (inside <script> tag)

<script>
  document.write("Welcome to JavaScript!");
</script>

External JavaScript (separate file .js)

Create a file: script.js

alert("Hello from external JS file!");

Link it in HTML:

<script src="script.js"></script>

4. How to Save JavaScript File

  • Save your file with a `.js` extension (example: `app.js`).
  • Use <script src="app.js"></script> inside your HTML file to connect.

6. Advantages of JavaScript

  • Easy to learn and use
  • Works in all browsers
  • Makes websites interactive
  • Large community support
  • Can be used for both frontend & backend

7. Disadvantages of JavaScript

  • Runs on client-side, so sometimes security issues
  • Different browsers may behave differently
  • Large code may slow down the browser
