In our daily life, we use symbols like +, -, *, / for calculations.

  • You add money to your wallet ‚Üí +
  • You subtract expenses ‚Üí -
  • You multiply items ‚Üí *
  • You divide pizza slices among friends ‚Üí /

In Python, these symbols (and more) are called **operators**. They help us perform actions on values and variables.

üîπ 1. Arithmetic Operators (Basic Math)

These are the operators you already know from school.

Operator Meaning Example Output
+ Addition 10 + 5 15
- Subtraction 10 - 5 5
* Multiplication 10 * 5 50
/ Division (decimal result) 10 / 5 2.0
% Modulus (remainder) 10 % 3 1
** Power 2 ** 3 8
// Floor Division (whole number) 10 // 3 3

üëâ Real-life example:

  • Adding prices of fruits in a bill.
  • Dividing total chocolates among kids.
apple_price = 50
banana_price = 20
total = apple_price + banana_price
print("Total Bill:", total) # 70

üîπ 2. Assignment Operators

Assignment means putting a value into a variable.

Operator Example Same As
= x = 5 x = 5
+= x += 3 x = x + 3
-= x -= 2 x = x - 2
*= x *= 2 x = x * 2
/= x /= 2 x = x / 2

üëâ Real-life example:

  • Bank balance increases after deposit ‚Üí balance += amount.
  • Game score increases after hitting target ‚Üí score += 10.
score = 0
score += 10
print("Current Score:", score) # 10

üîπ 3. Comparison Operators

Used to compare two values. The result is always True or False.

Operator Meaning Example Output
== Equal to 10 == 10 True
!= Not equal to 10 != 5 True
> Greater than 10 > 5 True
< Less than 5 < 10 True
>= Greater or equal 10 >= 10 True
<= Less or equal 5 <= 10 True

üëâ Real-life example:

  • Checking if your marks are equal to pass marks.
  • Checking if your age is greater than 18.
age = 20
print(age >= 18) # True
print(age == 18) # False

üîπ 4. Logical Operators

Used to combine True/False conditions.

Operator Meaning Example Output
and Both must be True (5 > 2 and 10 > 5) True
or At least one True (5 > 10 or 10 > 2) True
not Reverses result not(5 > 2) False

üëâ Real-life example:

  • A student passes if marks > 35 and attendance > 75.
  • You get free delivery if order > 500 or you are a premium member.
marks = 80
attendance = 90
print(marks > 35 and attendance > 75) # True

üîπ 5. Bitwise Operators (working with bits)

Computers store numbers in bits (0s and 1s). Bitwise operators work on these bits.

Operator Meaning Example (a=5, b=3) Result
& AND a & b ‚Üí 101 & 011 1
| OR a | b ‚Üí 101 | 011 7
^ XOR a ^ b ‚Üí 101 ^ 011 6
~ NOT ~a -6
<< Left Shift a << 1 10
>> Right Shift a >> 1 2

üëâ Real-life example:

  • Imagine two rows of light switches. & lights up only if both are ON.
  • << is like multiplying by 2, >> is like dividing by 2.
a = 5 # binary 101
b = 3 # binary 011
print(a & b) # 1
print(a | b) # 7

üîπ 6. Identity Operators

Used to check if two variables are the same object in memory.

Operator Meaning Example
is True if same object x is y
is not True if not same object x is not y

üëâ Real-life example:

  • Two books with same content are equal (==).
  • But if you‚Äôre holding the same physical book, that‚Äôs identity (is).
x = "hello"
y = "hello"
print(x == y) # True (same value)
print(x is y) # True or False (depends on memory)

üîπ 7. Membership Operators

Used to check if a value is present in a sequence (like a string).

Operator Meaning Example
in True if found "a" in "apple"
not in True if not found "z" not in "apple"

üëâ Real-life example:

  • Checking if your name is on the guest list.
  • Searching if a letter is inside a word.
word = "banana"
print("na" in word) # True
print("z" not in word) # True
